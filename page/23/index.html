新世界

发表于 2016-07-01   |   分类于 诗歌   |  

一湾清水
静静地流淌在沙土地下

撼动大地的血脉

古今世界的文明
纵横交错在图书馆的层层架子之间
心中的罗盘骤然指向
那广阔原始的非洲大陆

棕榈树的枝丫裁剪着自由的晚风
动物间的血腥与残酷是另一种美
涤荡在儿童微笑的目光里

返校随感

发表于 2016-06-26   |   分类于 个人随笔   |  

“过午，到图书馆加油。”曾经是再自然不过的事，今日如此却颇感心头五味杂陈。

毕业了，我真的毕业了，从小学开始直到硕士毕业，十八年，漫长的学生时代告一段落了。生活就是永恒的丧失，我们无时不刻不在丧失中活着。其实，惟因为我们曾经拥有过，才会有丧失的感觉，我们在拥有时珍惜了，也就无所谓后悔了吧。总之，混着泥土和青草芳香的校园时光已从我的生命中渐渐远逝，将会酿成我日后品味不尽的甘甜回忆。

今天是我第一次以校友的身份重返校园，虽然我在三天前才刚刚办完毕业的所有手续离开学校。但不知怎地，中午在勤业餐厅吃饭，往日麻木的味蕾好似突然间觉醒了，每一口饭菜嚼在嘴里，竟都是那样香，那样可口，香气直熏得我双眼模糊。

曾经，如今

发表于 2016-06-23   |   分类于 个人随笔   |  

念曾经，我在象牙塔的塔尖高歌，白云从窗外悠悠流过，寄托着我美好的心愿。我的遐想漫天飞舞，越过山河大川，掠过峭壁陡崖，万丈土地任凭我信马由缰，纵横驰骋。我追逐真理的灼热目光，穿透无尽冰冷的黑夜，照亮了遥远天际的几多星辰。我与闪电之神为敌，将巴别塔散落的砖块徐徐垒起，执着筑就通往远方的路。

叹如今，蹉跎岁月，我辜负了大好时光，背弃了年少梦想，成为了我曾立志不成为的人。呵，社会真的能教我们什么吗？是的，它教会了我在街头巷尾苟且偷生，它教会了我为五斗米而低下高贵的头颅，它教会了我浮夸与虚假，它教会了我只有放声叫嚣才能镇压欺软怕硬的，同是在市井大地上讨生活的，业已从社会大学毕了业的无奈的人们。悲夫！当虚伪战胜了真诚，当浮躁碾碎了宁静，当金钱扎破了梦想，我们空虚的灵魂里，还剩下什么？

念曾经，我的内心恰如满园春色，心中的红杏挥舞着想象的翅膀，迫不及待地飞出园子，自由翱翔，那天地似乎狭窄得远不够它飞行。

叹如今，荒芜占据内心，厚厚的白雪覆盖了曾经的热血与生机，肃杀的死寂枯萎了心园里的最后一根枝条。原来，我的心在宇宙间跳动得如此微弱。

念曾经，我喜动纵横万卷书，风雨兼程，夜半黄昏，不把分秒落。

叹如今，心若死灰，我开车在拥堵的街道，听广播里无聊的笑话。

念曾经，我也有过梦。

叹如今，我终日写诗。

曾经啊曾经，就是我内心死去的雄狮，何时能够浴火重生！如今啊如今，就是我虚空的躯壳，唯唯诺诺，蝇营狗苟，惟求生存而将自我丧失得杳无痕迹。形同此文，开头体态颇丰，结尾寥寥无言。曾经与如今渐行渐远，繁花落尽门庭冷。灵魂与躯体早已分离，逝者如斯空寂寥。

夜狂饮，问苍茫大地，何以自容？杯且住，梦里春归，携手泛舟去。

搬家记（一）

发表于 2016-06-13   |   分类于 个人随笔   |  

独坐在家中书桌前，耳畔仿佛还听见窗外传来的阵阵敲钟声。硕士三年，转眼已要剧终，我的心情是复杂而又简单的。说它复杂，是因为心中五味杂陈，不可言说。说简单，是因为日子依然还是会一天一天平静地继续下去，时间不会因什么事而略微迟疑停顿。

昨晚，我开始整理宿舍了，室友说真是到了感伤的时刻。是啊，相聚分离，转身千年。昨天我们初识，明天我们会在哪个路口相遇，我们又会在哪个餐馆重逢呢？

我要收拾的东西很简单，大致分为两样：书和其他生活用品。首先是收拾书，我用七八个能用得上的袋子、纸盒，还有行李箱，利用各种缝隙将它们塞得满满当当，总算是把我“默默堂”三分之一的书给收拾好了，今天便运回家中。

咦，你可能会好奇，“默默堂”为何物？这里有段趣话：某日，我和梦梦坐在图书馆看书，我小声和她说历史上的许多著名作家和学者都会给自己的书房起个名字，大多为“某某斋”、“某某楼”、“某某轩”、“某某阁”、“某某堂”，诸如此类。我提议给我自己以后的书房起名叫“默轩”，问她意下如何。梦梦思忖片刻，笑嘻嘻地说：“你说叫‘默默堂’如何？”此为“默默堂”首次以文字方式出现，而我又觉有趣，便沿用下来。

随感

发表于 2016-06-05   |   分类于 个人随笔   |  

今日过午，天色阴沉，果不久就听窗外暴雨倾盆。下雨天就是适合在寝室呆着，读完了季老的《清华园日记》，眼皮坠地，遂蒙头大睡，四时始醒，见天已转晴，提上枪包出门挥杆。

下午挥杆手感颇佳，7号铁频频打出140码以上的距离，心情大爽，最后一球更是以完美的弧线飞进夜色朦胧中，稳稳落在150码的牌前，目测约摸有148码了吧。这周末挥杆的心得就是，集万念于一心，集全身力量于杆头击球之一瞬，如此始能把球击出一满意之距离来，心中亦颇感快慰。

除了挥杆，今天我对于老年人还有一新感想。以前对于老年人，我总是尊重与同情并存，有时甚至同情的成分更甚。可是，今天我觉到其实不必同情老年人，只要尊重和力所能及地帮助他们就好，因为他们也有过年轻灿烂的日子，因为我们也有老去的一天。每个人的生命历程其实大同小异，我们身上的喜怒哀乐也曾在他们身上出现，我们的烦恼、开心，他们也曾经经历过，不同的只是，仅仅只是我们所处的年龄段不同，阅历有先后而已。
