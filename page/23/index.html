下厨小记

发表于 2016-08-28   |   分类于 个人随笔   |  

这个周末过得自娱自乐，间或与本科室友和高中好友聊聊天，轻松惬意。

昨天早晨心血来潮，醒了躺在床上用手机打开唱吧，起初是听歌，听着听着，自己也唱起来。于是起床后，在家里放声歌唱，用唱吧录了两首歌，《彩虹》和《演员》。好长时间没录歌了，虽是两首歌，不过竟唱了几个小时，好不欢乐。我平时说话少，这一唱也着实痛快，还感觉自己的声音听起来挺有魅力，单曲循环了一天，哈哈！

今天起来，吃了个简易早餐，稀饭、肉松、豆腐乳，在家吃自在，也卫生。然后顺应自己的心情，在书架上挑了本梭罗的《瓦尔登湖》读起来。下午在家里练习练习推杆。傍晚时扫扫客厅、扫扫厨房、擦擦房间的木地板，刷刷马桶，心情大好。接着，在厨房忙活起来，这还是我头一次自己给自己做顿晚餐吃哩。

洗米、煮饭、洗菜、打鸡蛋、炒菜、起锅、关火关煤气、狼吞虎咽大吃一顿、刷碗！一切井井有条，做起来也很有意思，一点儿不感到麻烦。吃起来更是感到饭菜都倍儿香！

从小就耳熟能详的那句“自己动手，丰衣足食”，怎料实践起来竟拖到了二十五岁，想来惭愧。感恩二十多年来为我做饭的人们，是你们养育了我啊！工作了以后，我希望自己勤快一点儿，不再为生计苦恼。

鸭汤

发表于 2016-08-21   |   分类于 诗歌   |  

夜已深沉
我在灯下品读泰戈尔的小诗
母亲轻轻推开门问道
要不要喝点儿刚炖好的鸭汤

我摆摆手说快要睡了今晚不喝
母亲在门外坚持着
喝一点儿吧
我点点头

门外传来父亲的脚步
我叫声爸爸
说我还是不想喝
你让妈别张罗了

隔着门
我却能看到母亲眼角的失落
那垂下的双手
自顾自地哼起G弦上的咏叹

我的目光错落在字里行间
黑压压的文字好似成群乌鸦
这静夜的碎片
纷纷扬扬

母亲轻轻推开门说就喝半碗好吗
我使劲点头
乌鸦顷刻全没了踪影
唯留下一纸光亮

诗的形式

发表于 2016-08-15   |   分类于 诗歌   |  

我再也没有遇见
比这更无聊的下午了

演讲者简直是台催眠机器
毫无起伏变化的语调
毫无情感的流露
年轻的生命们昏睡了整个下午

我在催眠机播放的间隙
读木心的小诗
一股清泉流过心间
像是解药

成堆的文字
被残忍地堆弃在无厘头的材料中
这材料没有丝毫生气
语言也失去了应有的色彩

时钟的指针拖着那灌了铅的脚
越走越慢
杯中的水眼看就要见底
怎奈催眠机依旧电力十足

半梦半醒之间
终于蹦出的一句谢谢
宛若空谷里一声惊天的枪响
杀死了这个无聊的下午

一个月的集训生活

发表于 2016-08-14   |   分类于 个人随笔   |  

昨天，是集训营结束的日子，584人“刑满释放”，这其中就包括我。

集训从7月15日报到开始，到昨天8月13日收拾行李离开。集训的地点在厦门双十中学枋湖校区，校区不大，但风景宜人，和厦大校园在细微处挺相似，可能是同在厦门的原因吧，建筑和树木看起来都不陌生。

在去集训营的那天白天，我写了首诗《囚鸟》，感慨自己即将拥有的自由变窄了。我们每个人都有自己舒适的个人空间，在家里空间大，住习惯了，自然也越感到自由。而集训营是五人同住一间，而且彼此初次相识，想来自己的个人空间必然少之又少了。其实，我无意去解读自己曾经写下的诗，我总觉得，写诗时固然有着当时的思绪和心境，但随着时光的流转，一段时间后再读以往自己写的诗，不必太用心去回忆当时的所思所感，而只是读诗，只是感受文字沉淀下来的韵味就好了。“回首向来萧瑟处，归去，也无风雨也无晴”，往事既然已经过去，又何必那么刻意地想去抓住些什么呢？唯品味由生活的佐料酝酿而成的诗，品味诗中历经时空变迁的字句，品味去除了杂质而留下的最本真的韵味，由此我们获得了单纯的美的享受，还有比这更好的记录生活的方式吗？

发表于 2016-08-08   |   分类于 诗歌   |  

一段提前到来的幸福时光
我们与七夕邂逅在周末的阿罗海

正如生命原本没有名字
日子也只随你我的心境而流转

备忘录中那沉静多时的纸的时代
在高高的书架之上苏醒

梦露歌剧院的放映室里
我横刀立马挡在妖魔与你之间

水面上漂浮的七彩泡泡
无邪地注视站在一旁遐想的你

金色波光里矗立着大海忠诚的灯塔卫士
风筝凌空俯瞰沙滩边嬉戏的狗和人

游乐场里的碰碰车迸出了几代人欢乐的火花
投篮机前我们洒下兴奋的汗水

你千辛万苦总算捕捉到一个表情正常的合影瞬间

动车载着你驶往家的方向
日记的惊喜还在我脑海里快乐蹦跶

带着淡淡甜味的回忆从心底缓缓升起
此刻我口中的咖啡也愈加温润香醇

放风——与你一起的周日

发表于 2016-07-31   |   分类于 诗歌   |  

工作日被占得满满当当
周末重又值得期待
自由的灵魂
总是早早地叠好被子哼着小曲儿出门

围墙与校门外边是另一个世界
阳光都变得轻松与欢快
风轻轻吹动绿叶
唤醒漠然沉睡了一周的心

芙蓉湖畔高耸的主楼
勤业餐厅丰盛的早餐
丰庭宿舍区的旧铁门
都氤氲着浓浓的家的味道

小屋有你美美的笑脸
阳台上晾着的衣服翻飞起舞
午后咖啡的香气充盈着四周
书页在静谧中缓缓掠过指尖

你轻触键盘的声音
宛若美妙的乐曲
我们看时光的影
从桌子的这头渐渐滑到那头

泛红的夕阳依旧如此鲜艳
光线穿透枝叶的罅隙
亲吻着你的脸颊
噢，你的眉眼俊美如诗

晚风从湖上拂过
你飘起的缕缕长发
惊艳了桥边的野花
也牵动着我的心

与你一起散步
看这城市的车水马龙
路边的仙人掌
仿佛将我们引入新的境地

白鹭洲公园里琴声悠扬
灯光应和着起舞
我悄悄问你
是不是天上的星星也能听见这场音乐的盛宴

巴黎序曲在理查德的指尖流淌开来
我们仿佛又置身塞纳河畔
依稀的树影后面
是圣母院精致的侧脸吧

晚饭后一分为二的纸巾
步道上牵你温热的小手
幸福不随时空而变得不同
只要你在我的身边

公交站台旁
静静目送汽车载着你缓缓离去
我向霓虹灯的尽头挥手道别
期待与你下一次的相见

重逢

发表于 2016-07-18   |   分类于 诗歌   |  

乍到新处
颇觉物非人是
张张面孔何其相似
恰似老友重逢
我猜，那是同样的灵魂附到了相似的肉体上
亦或是人的长相本来就很相似？

时光在校园的操场上接力奔跑
我带着自己和自己的爱好
听着老朋友霍洛维茨弹奏的钢琴曲
一次次成为陌生境遇的主人

人生的场景虽不断变化

藕断丝连中变着戏法继续着

我们常说每次相见都是初见
而生命的初见竟也会有这般重逢之感
重逢的亲切与喜悦
久久盘踞在我的心间
