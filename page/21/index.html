入夜

发表于 2016-11-27   |   分类于 诗歌   |  

夜幕遮盖在城市高耸的楼宇间
西天金色的余晖
一点点被驱赶
直到一扇扇明镜似的窗格子角上
消失

书桌旁透明的落地窗紧闭着
却无法阻挡泼墨似的黑暗
大团大团地闯入
这夜的精灵在我身后聚集
他们的庆典即将开始

噢，不，这是我的房间
请把你们热闹的乐声调小些

我伸手打开桌上的台灯
昏黄的光立即投照出一小块静谧的空间

感恩节随笔

发表于 2016-11-24   |   分类于 个人随笔   |  

厦门真可谓是一夜入冬，前两天还穿着短袖开着空调，昨晚一下子气温就降下来，让我深感冬天已朝这座海滨城市呼啸而来。此刻的我，坐在家中便穿上毛绒绒的暖和的“熊熊衣”了。

今天是感恩节，又恰逢发了点儿薪水，我便把其中的一部分用来在微信中发红包给我亲爱的家人。发的时候感慨，微信常常翻来翻去，但是在闹腾的世界里最值得用真诚的爱和感恩的心来对待的，只有最亲近的这么几个人而已。世界很大，但我最亲近的人真的不多。有你们相伴的日子，我的生活是多么开心与幸福！

断绪

发表于 2016-11-19   |   分类于 诗歌   |  

咖啡粉
扬扬洒洒从包装袋中抖落
与灼热沸腾的开水汇合
一杯午后的期待便诞生

大吉大利
茶几上一个红包正闪着金光
既然话语终究要消散风中
为何还要吝惜美丽的言词

我在乱纷纷的拼图碎片里
找寻老去的时光
纵使那每一片都满含情愫
可我却仍旧拼凑不出童年记忆的全貌

高高的楼里住着许多人生
他们在不同轨道上奔跑
温暖明亮的光
给每个阳台上的衣服都涂抹上艳丽色彩

今天是哪一天了
这日子静默一如从前
秒针悠悠的
远方的风儿越过山冈

安静地过个周末

发表于 2016-11-13   |   分类于 个人随笔   |  

好长时间没有写篇像样的博文了。最近博主跑哪儿去了呢？这确实是个好问题。我也在想，最近我去哪儿了呢？哦对，最近急急忙忙地奔波在厦门这个城市的大街小巷，看遍沿路的红绿灯和斑马线，工作讨生活去了。工作日按时上班下班，白天几乎都是待在一间小屋子里面对着电脑屏幕度过的，两耳不闻自然之声，但闻隐约起伏的机器轰响之音。事儿多时连续几天加班加点干活，没啥事时就在中午小憩之后泡上一杯香醇可口的咖啡，悠哉地啜饮起来。而且常常咖啡味一瓢入鼻孔，我的心间就溢满了诗意，偶尔打开文本编辑器顺手便写成一首不知名的小诗，自娱自乐一番。每天中午小憩前读几首Holderlin的诗，也为工作日增添了几分惬意。

工作总体上来说还是顺心自如的，主要做的是和数据的分析、加工、处理相关的内容，难易度不定，简单的就自己处理，处理完让导师或者熟练的老员工帮我检查核对一下，难的话，就四处请教，直到弄明白处理好为止。工作中最主要的还是与同事的有界线的友好相处，冷热度适中，不卑不亢地以完成工作任务为核心，做好每件小事，过好平凡的每一天。

夜湖吟

发表于 2016-10-12   |   分类于 诗歌   |  

微风轻轻吹过
在湖面留下细碎的印迹
明灭浮动
是风的影子

黑天鹅浅诵夜的絮语
这自由之声只属于它们自己
道道乌黑的魅影
是夜里最灵性的流动

是什么在我的脚边打转
噢，一只硕鼠
黑夜让你的目光更加敏锐
湖岸是饱餐过后惬意的闲散地

黑夜，你多么伟大
万物在你面前褪尽它们光鲜的色彩
唯剩了静默的黑与白
难道这才是万物的本质之色吗？

树丛里阵阵虫鸣啾啾
耳朵恢复了听觉
眼前清醒的朦胧
是我双眼新生的视力

枝条俯身亲吻着湖面
身旁的人来了又去
我久久地在湖畔静坐
凝视着无语的苍穹

不知瓦尔登湖的夜晚
与今夜会否相似
身后树林深处
许是也有过那纯净的木屋吧

水波在夜风中翩然起舞
深不可测的湖啊
且把你无尽的故事
慢慢说给我听

2016.10.12于芙蓉湖畔

今何往

发表于 2016-10-03   |   分类于 诗歌   |  

我将一枚银闪闪的硬币
投进了啡派咖啡机便便的大肚子中
一杯浓咖啡便穿过重重记忆之门
把熟悉的温度传入我的掌心

推开吱呀作响的木门
午后暖黄色的阳光充盈着教室
地板在这一刻的静谧中苏醒
好像认出了我似的咿咿呀呀地不知在诉说着什么

我回到原位
仿佛灵魂重又回到多年前的自己身上
我久久地坐着
一切都像极了午后的一个打盹儿

猛然间我抬头四望
课桌椅依旧静默在青葱的时光里
只是那些我熟悉的人啊
你们如今都去向了哪里

我呆呆地凝视桌上的咖啡
迟迟不敢品一滴入口
生怕如今的味道
已不复如记忆中那般醇香

2016.10.3于学汇楼506
