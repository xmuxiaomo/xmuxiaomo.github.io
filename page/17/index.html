另一种想念

发表于 2017-04-04   |   分类于 诗歌   |  

那颗星在动
纵使我屏息静气地坐定仰望
可分明还是觉得它在动
我轻轻哼起一段旋律
星星那微弱的光点也随之乐动

你也喜欢这支歌吗？
这支小时候爷爷奶奶哄唱给我的歌
为何你闻之而洒下一颗光的泪珠
没有名字的星啊
也许你就是我的亲人吧

我并不时常像今夜这样注视着你
但你时刻守护我
在数不清的光年以外
我问你在无边的黑暗里是否感到孤独
可我听不见你的回答

我越是凝望星空
便发觉星星越多
倬倬约约深深浅浅的星噢
你们是我祖祖辈辈的先人吧
我回应以心底的一束微光

有人说时常仰仰头
可以治好颈椎病
满天的星星都眨着眼大笑
我也跟着大笑
我们约定下个夜晚再见

2017.4.4 清明节
于上弦场

亮剑

发表于 2017-03-30   |   分类于 诗歌   |  

那持续数百秒的哭喊声如此凄厉
在我的心头缠绕成麻
无法开解

我一刻也不能停在原地无所作为
任何多余的念头都是对生命的不敬
冲锋是唯一的号角

我的行动不由自主
身体里的热血早已沸腾
荣耀与骑士精神指引我向前

和平的蓝天下
罪恶在潜行
可是我身边的人呐，你们却为何无动于衷！

我要致敬，向那些百战沙场的勇士们
高贵而无所畏惧的心啊
纵然在战斗中失去生命，又何足可惜？

此刻，沉默终于不再沉默
它高声疾呼——
别让哭喊的人感到绝望

年轻的答案

发表于 2017-03-27   |   分类于 诗歌   |  

我们的生命原本偶然而没有目的
我们于迷茫中努力为自己的生命寻出个意义
我们研读哲学，也求诸科学
终于还是困惑不得其解

我们每日重复着工作与生活
多像神话里的夸父和西西弗
一个执意追逐着太阳的方向
一个于山坡上永不停歇地推动巨石

维特根斯坦说：人生问题的解答，
在于对这个问题的消除
当我们停止追问“什么是放下”的时候
我们突然间领悟了放下

求自在不自在知自在自然自在
想如来念如来悟如来如是如来
我不赞成生命是虚无与荒谬的
却也无法否认生命的荒谬与虚无

既然我如此幸运地降生于人世
又有何理由不尽情探索与经历一番？
我只愿如其所是地生活
不负宇宙间这份美妙的馈赠

2017.3.27
于总行

蔚蓝天空下
古镇小道旁
城墙将影子投在斑驳的石砖上
北宋的院落格外清净

读《阿古利可拉传》小感

发表于 2017-03-23   |   分类于 读书笔记   |  

用两个晚上的时间阅读了古罗马塔西佗的作品《阿古利可拉传》，心中感悟良多。我记不住书中提到的许多帝王将相的名字，但对阿古利可拉的中庸之道，对其行事的谨慎与谦和是无法忘怀了。

书中有这么一句话：“在那个时代，美誉和恶名有着同样的危险。”的确如此啊，自古以来人类的天性里就有嫉贤妒能的特点，恶名者被世人唾骂是很常见的，但是美誉为何危险？就是因为享有美誉之人周围不乏小人，小人心胸狭小，见不得别人的好，阿古利可拉在任期间，为罗马攻占不列颠立下不朽的功勋，但是当时的皇帝多米先就处处堤防与担忧，害怕大臣的光芒与荣耀高过自己。不过阿古利可拉实在是谦和君子，他的穿着与外貌及其普通，引不起身边之人的注意，丝毫没有立大功者的自豪与骄傲，这也是他为人处世的大智慧之所在啊。多米先是想挑刺也挑不出来，只好派人多加注意阿古利可拉的行动而不敢再有所动作了。

不列颠方面，一位名叫卡尔加库士的酋帅在集结兵马、号召天下勇士以抵抗罗马的进攻时，作了一番振奋人心的演说。演说以自由为最高宗旨，不愿为奴的精神在听众间引起巨大的反响。这篇演说非常棒，我想把它摘录在此处：

周日小记

发表于 2017-03-19   |   分类于 个人随笔   |  

又到美好的周日时光，我和梦梦依旧出发去“翔安度假村”。最近这几个周末，我们时常会在游客挤满思明校区的时候，悄悄出逃，自驾前往翔安校区，那儿地广人稀，白天在校园里几乎看不到人和车，很有到了另一个国家甚至另一颗星球之感。大大的校园里，仿佛只有我们两个人，阳光明媚也好，小雨点点也好，都十分惬意舒畅。

周末，我们喜欢换个环境生活一天，距离的拉远，让我们远离日常的琐碎事务，让周末时光更加纯粹与静谧。在食堂三楼吃碗不辣的麻辣烫或一丁点冷辣的凉皮，在校内的万达影城看个新上映的电影，在图书馆的自习室安静地看书写作，无一不让我们感到开心与愉悦。

今天中午吃过午餐，我和梦梦说：“你将面临一个严峻的考验”，梦梦眨巴着大眼睛问我是什么考验，噔噔蹬蹬——练车！坐上车，调好座椅和后视镜，我便爽爽地坐上副驾驶座，当起教练来。其实梦梦拿驾照的时间比我还早一年，只是后来开得很少。不过车技学完一般也不会遗忘，操练操练，但手熟尔。梦梦开得很认真，我坐得也很放心，她说想练习一下沿环岛绕圈，我说好啊，结果梦梦练得起劲，一圈接一圈，直到我头晕目眩，她才笑嘻嘻地从一个出口转出去。

手机

发表于 2017-03-19   |   分类于 诗歌   |  

手机此刻正握在我的手里
我要用它来赞美它自己
五十年前的人们想不到它今日的普及
那时它甚至只是渺远未知的概念

空气是无形的介质
我们看不到被叫作信号的东西
邮递马车在斜阳下驮着书信蹒跚远行
书信却在指尖轻盈迅捷地流转

亲人挚友那远自故乡的问候
即刻便直抵我们心间
时空因手机而扭曲

我放下这神奇的小物件
推开窗子
悲伤与喜悦布满了整个天空

2017.3.19
于翔安校区图书馆
