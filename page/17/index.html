辗转

发表于 2017-03-10   |   分类于 诗歌   |  

思想的笔在脑中咆哮
白纸的荒漠需要词语的墨水

时间重重压住我的双肩
脖子渴望舒适的倚靠

我用清水搓脸
才恍然发觉脸皮是如此之薄

虚无在白昼里蔓延
脊柱忠实镇守身体的花园

雨解开了天空的扣子
为给大地织一席锦衾

我吐出的一小团叹息
旋即凝结在床头

夜在枕上沉沉睡去
而我仍在追赶梦的列车

2017.3.10

天空

发表于 2017-03-04   |   分类于 诗歌   |  

在暖意尚未抵达的三月初
我喜欢看傍晚六点钟的天空
太阳收敛起耀眼的光芒
天空只留下淡淡的黄蓝色光亮
我还看不到群星
因为天空还没换上夜的披风

天空很忙
在轻飘飘的云的上方
是一道道飞行的航线
天空很高
有飞鸟也无法越过的空间
天空深浅不一
有比浅更浅的远方
也有比深更深的遐想

我听不见天空的语言
也不知道此时有谁和我一样在望着天空
会是游吟的诗人吗？
会是沉默的垂钓者吗？
会是醉眼看世界的乞讨者吗？
会是远走天涯的浪子吗？
会是仰头问天的青年哲学家吗？
又或者，是一个穷困潦倒的画家吧

风将云层吹得紧密起来
大地已华灯尽染
我喜欢看傍晚六点钟的天空
我享受天空赠予我的无尽幻想
我大概能够理解天空
因为我拥有和天空一样广阔的胸怀

2017.3.4
于十一楼阳台

甩锅文化

发表于 2017-03-04   |   分类于 个人随笔   |  

夜里关了手机，一宿无梦，睡到自然醒，醒来一切都那么安静，好久没有这般舒爽感受了。近来大多数时间都是闹铃将我叫起，噢，又该上班了！过去的这二十天里，由于公司项目赶进度，上了十七天班，只休息了两天，今天是第三个休息日，而且其中大概有十天晚上都加班过了九点，实在是疲惫不堪。回家倒头就睡，上班久坐不起，日子久了，头昏脑涨，并感觉腰部有点疼痛。

确切来说，不是有点儿疼痛，而是年后到现在几乎每晚睡觉都会痛醒。是福是祸，这说不准，爸妈比较担心，说让我有空去医院检查检查。我倒觉得这是个“好毛病”！大概每晚睡到六个小时左右，就会开始疼痛，这样既有了之前平稳的睡眠，又可以早起，而且这毛病也怪，起来一会儿人就好好的了，没有任何不适感，就睡觉时才有，也许是命运之神在提醒我，该早起啦！

哈哈，言归正传，扯了一堆闲话，该说说近日学习到的一种新公司文化了，我将它取名为“甩锅文化”。顾名思义，就是将背在自己身上的锅丢给别人，锅且不论黑白，只要一接到手上，迅速丢出去，自己一身轻松。起初培训时听了不少介绍公司文化的讲座，独独对此“甩锅文化”没有耳闻。为何我近日发觉到了呢？如你所想，我定是有许多亲身体验的。

巷弄

发表于 2017-03-01   |   分类于 诗歌   |  

沿不太陡的小径往上走
渐渐远离闹市
脱去嘈杂的外衣
脚下便有轻风

旧宅沐浴和暖的阳光
电线胡乱地搭架
蔬菜刚刚入锅
窄窄的小巷幽幽为我带路

曲径狭长望不到尽头
不知其通向何处
唯闻钢琴声星星点点
仿佛溪水静静地流淌

每每迈步向前
儿时老家的记忆就愈发清晰
老家近旁也有窄窄的小巷
黄昏是一支金色的歌曲

画笔用心调配大自然的色彩
远山深浅黑白影影绰绰
羽毛球飞上落满霞光的屋顶
再不愿醒来

日记本

发表于 2017-02-26   |   分类于 诗歌   |  

夜晚读书写作时
书桌上总有你最甜美暖心的微笑
陪伴我陷入深深的迷惘
也陪伴我写下对生命的每一次感悟

翻看日记本
无意间发现
原来你是我诗意的启发者
你将优美的诗与我分享
那潜藏的诗歌的种子
不经意中已在我的心里长出小小的嫩芽

读着曾经写下的我们之间的点点滴滴
回忆浸湿了我的眼眶
那是美的令我思念的感动
往事那么真
一幕幕在字里行间漾开
一笔笔流淌出心灵的无限喜悦与沉醉

你说感谢我记录下这许多美好珍贵的瞬间
可你却不知道，正是你啊
我亲爱的恋人
是你，让我把生活过成了诗

我奢想，我有我们全部的记忆吧？
哦不！我只拥有一半的记忆
还有一半该在你那儿
又或是
我们都拥有全部的记忆
关于我们爱情全部的记忆

这日记是秘密的
这是只属于我们两个人之间的秘密
可我担心，它终有一天会成为公开的秘密
有另外一颗心灵在另外一个时空里
开启这岁月的封条
那时，一定有轻柔美妙的乐曲从匣子里潺潺流出

2017.2.26
于家中

哈哈镜

发表于 2017-02-24   |   分类于 诗歌   |  

镜子里的人矮了
大盖帽险些触及豆浆
我将脸蛋埋进小碗
奇怪，我的椅子被谁弄翻了？
我哈哈大笑摔个四脚朝天
却惊奇地发现周围满是
同样的
齿牙未全的大笑

镜子里的人胖了
两张没写完的作业纸趴在桌上沉沉入睡
阳光越过窗沿
梦被揉碎在我的双眼间
脚踏板与上课铃一同涌进教室
寒冷枯枝旁
与我无尽奔跑的
是笔尖下不竭的蓝色墨水

镜子里的人瘦了
玫瑰花如秋风般散落
我的心是一片海
有海鸥和气球在飞翔
海边是宽大的广场
小烘焙坊藏不住咖啡的香气
我的鼻子像一把镰刀
不住裁剪这味道

镜子里的人高了
胡渣洋洋洒洒飘下
我望向自己深邃的眼眸
噢，我的身体哪儿去了！
领带和西服在空气中跳起恰恰
可我分明还感觉到我的存在
哈哈镜笑了
哈哈镜和我一起碎了一地

2017.2.24
于海滨六楼
