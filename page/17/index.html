五里桥

发表于 2017-04-18   |   分类于 诗歌   |  

望高楼还在守望熙攘的人潮
八百多年了
这街市依旧热闹非凡

黄昏苍老的皱纹间挤出几许暖意
微醺的芦苇齐刷刷望向风的背影
安海上粼粼的水波浮光跃金

五里桥此刻就延展在我脚下
古老石块已然被行人的步履打磨得光亮
石面凹凹凸凸将记忆深深浅浅纳入我的足底

噢，是谁在拨弄风的琴弦？
如此动听的田园赞歌无声流淌
令时光闻之落泪

我打开日子的收纳盒
始终翻找不出比此刻更惬意安宁的时辰

沉思杂感

发表于 2017-04-13   |   分类于 个人随笔   |  

我又再次深陷虚无与困顿之中，我们给自身所加的限制是否太多？我们除了今天的生活方式就没有别的了吗？我们是自己真正在生活着，还是从来都只是太过于懒惰而选择盲目跟随既有的模式来生活？我们真的能安心地合群生活，活成别人眼中所期待的样子吗？

忙忙碌碌的每一天，每个人行色匆匆，与时间相互追赶，把新鲜的一天天过成了单调重复的同一天，真有什么事儿是非做不可的吗？三十米外的两个人在无意义地争吵，争吵的根源只是情绪的波动罢了，一时兴起，为逞话语的快感而喋喋不休没完没了地争执，你们俩还不如直接决斗来得干脆！

我因无法忍受争吵的噪音而从室内的角落挪坐到了大堂落地玻璃外的石阶上，客气的保安迅速过来请我挪到十米以外的石头上去坐着，也许是担心我这么一坐，有损这耸入云霄的高端写字楼的形象吧。我无心与他争辩什么，因为这并没有什么意思，我只怕多说无益的话而扰乱了此刻内心的思绪。

儿时不读书，嬉闹贪欢有何不可？大家都是几十年的人间之行，一样的。没必要拘谨畏缩地生活，人往来于天地间，自在如一阵灵动的风。没来由的规定与制约几乎都是外来的，我们何必不清不楚地一味将就于它们？它们是否经得起质问与怀疑？我们是否真的知道遵守规定这种行为的最初动机到底是什么？

追

发表于 2017-04-10   |   分类于 诗歌   |  

时钟溜过八点
我急速跳上一辆公交
晴朗的天气洋溢假日的氛围
厦门一蹦而进入到夏天

街上的车流像饿极的狼群
竞相撕扯时间的残骸
我们是公交这匹老狼的果腹之物
老狼大可怡然自得地悠哉前行

我耐不住时间的落差
一挂瀑布悬隔了心灵与双眼
小皮鞋夺门飞跑
终究敌不过异化出轮爪的饿狼

唉，又迟到了！
时间笑我徒劳无功
我却欢喜地弹飞鞋里沙子的叛徒
用一身的汗水来赞美夏天

二十六岁
我和我的兴趣一同生活

出门尽情驰骋
在家坐拥书城
青草地上自在挥杆
音乐厅里聆听盛宴

亲吻阳光下每一颗飞舞的尘灰
奢侈放荡地生活

另一种想念

发表于 2017-04-04   |   分类于 诗歌   |  

那颗星在动
纵使我屏息静气地坐定仰望
可分明还是觉得它在动
我轻轻哼起一段旋律
星星那微弱的光点也随之乐动

你也喜欢这支歌吗？
这支小时候爷爷奶奶哄唱给我的歌
为何你闻之而洒下一颗光的泪珠
没有名字的星啊
也许你就是我的亲人吧

我并不时常像今夜这样注视着你
但你时刻守护我
在数不清的光年以外
我问你在无边的黑暗里是否感到孤独
可我听不见你的回答

我越是凝望星空
便发觉星星越多
倬倬约约深深浅浅的星噢
你们是我祖祖辈辈的先人吧
我回应以心底的一束微光

有人说时常仰仰头
可以治好颈椎病
满天的星星都眨着眼大笑
我也跟着大笑
我们约定下个夜晚再见

年轻的答案

发表于 2017-03-27   |   分类于 诗歌   |  

我们的生命原本偶然而没有目的
我们于迷茫中努力为自己的生命寻出个意义
我们研读哲学，也求诸科学
终于还是困惑不得其解

我们每日重复着工作与生活
多像神话里的夸父和西西弗
一个执意追逐着太阳的方向
一个于山坡上永不停歇地推动巨石

维特根斯坦说：人生问题的解答，
在于对这个问题的消除
当我们停止追问“什么是放下”的时候
我们突然间领悟了放下

求自在不自在知自在自然自在
想如来念如来悟如来如是如来
我不赞成生命是虚无与荒谬的
却也无法否认生命的荒谬与虚无

既然我如此幸运地降生于人世
又有何理由不尽情探索与经历一番？
我只愿如其所是地生活
不负宇宙间这份美妙的馈赠

2017.3.27
