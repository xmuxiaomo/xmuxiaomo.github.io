懒惰的分针

发表于 2017-07-09   |   分类于 诗歌   |  

我凝视表盘上的分针
它走得无比缓慢
我猜一定是在周末的下午
它才如此懒散地踱步

我侧着耳朵听时间行走的声音
可惜只听到不远处工地上传来的敲敲打打
我在敲打声里找寻拙劣的韵脚
飘荡的粉屑鸽群般升上蓝天

一个白色的塑料袋在天空飞翔
御风而行的样子多像晴朗的鸟儿
我发现窗外的世界是垂直的
所有的车都过山车一般极速坠落

魅蓝的天穹和表盘上的分针一样
动也不动
盛满绿茵与玫瑰的光华大地
却一眼万年

噢，狡猾的分针！
趁我发呆的间隙
你哧溜烟儿
颠倒了整个世界

2017.7.9
于家中

山雨

发表于 2017-07-05   |   分类于 诗歌   |  

山中的雨很安静
静悄悄地下个不停
起初没什么雷声
雨幕淅淅沥沥

屋里的人们安静地熟睡
姿势各异
偶然间一瞥
嘴巴都半开着歪向脸的一侧

雨下大了
雷声在远山的那边怒吼
桌上小球乒乒乓乓
我看不见雨里的房子和大山

翠绿的万物正畅怀撒欢
雨却踮起脚逃开

二姑夫

发表于 2017-07-02   |   分类于 个人随笔   |  

（一）

生活中，我与二姑夫的交集并不算多，我们见面的次数也不多，有时三年见一面，有时五六年才见一面。按说我们并不太亲近，毕竟在一起相处的时间少，共同话题也少。他在我印象里是个头脑简单的硬汉形象，肌肉发达，生活简单。

我不知道他是做什么工作的，不知道他的联系方式，不知道他原先的家庭境况，甚至很迟才知道他的姓名——林瑞清，一个充满了陌生感的称呼。

我唯一记忆深刻的是，小时候，估摸还不到十岁，有段时间我待在福州，住在二姑姑家里，他每天中午都来姑姑家，给我们做可口的饭菜。他知道我爱吃花蛤、小鸡腿，于是每顿午餐都少不了这两样美食。他烹饪的手艺很棒，深得我心。小鸡腿很香，我总是一个还没啃完，眼睛又不自觉地往盘子里看，他就笑着叫我再吃，我也毫不客气，一个接一个吃，直到撑得不行了，才放下筷子，他便把没吃完的小鸡腿留着，到晚上再给我吃。那时我不懂事，见他每天都来二姑姑家，洗菜、做饭，等我们饱餐过后，他又洗碗、收拾餐厅厨房，然后待一阵子又外出做事。我就问二姑姑：“他是你们家的保姆吗？”二姑姑笑笑没说什么，我说：“那他就是男保姆了！”

因为那时他还没有和二姑姑结婚，某天偶然在饭桌上，我叫了他一句“男保姆”，他愣了一下，笑笑没说话，让我吃菜。

闹中取静的一方园地
庭院式建筑古朴清幽
水声潺潺
鸟鸣啾啾

白云棉花糖似的静止在低矮天空

我游目四望
圣贤广场上学者们相谈甚欢

独自在回廊里吟诗

我行止于古今
不再问归途

爱车情怀

发表于 2017-06-15   |   分类于 个人随笔   |  

今晨，半梦半醒间，脑中闪过“驾驶员年龄不得超过60岁”这行字，再念及父亲今年已将近58岁了，不禁愕然惊醒，光阴匆匆啊！

父亲很爱车，也很喜欢驾驶。前年春节前，父亲买了心心念念已久的BMW X3，爱不释手，欣喜之情自然流露出来，即使有时没开车，只佩带着宝马钥匙扣，也会心中充满自信，满面春风。父亲时常一本正经地问我：“看看老爸像不像开宝马的？”我就笑嘻嘻地使劲点头，说：“哈哈，爸爸还没缓过神来，都已经开宝马了，还天天问像不像开宝马的！”于是我俩就哈哈大笑一阵。

去年我毕业开始工作了，以往在学校来来往往常见的是同学和老师多，自行车多。而工作后，家人将家中最早使用的一辆小骐达交给我开，以方便上下班和日常出行。所以这半年多来，由于自己开车，成天行驶在马路中央，眼中所见来往川流不息的车子就多起来。看见得多了，便自然会留意与关注，于是我也爱上了看车，也喜欢有事没事就出去自在驾驶一番。

记一场雨

发表于 2017-06-13   |   分类于 诗歌   |  

那座晴朗的岛
消失在迷蒙的暴雨中
只一刹那
天空挥动衣袖

雨点好像落花
在风中狂舞

檐下灯明
城市却很暗

雨声轻裹哀伤
和着肖邦的乐声
嘈杂失守
大地被静默占领

围观这场雨
人们欣喜地睁大双眼
反常的新鲜感
像盛宴等候多时

秩序的大厦倾塌
喜感有些莫名

记一场雨
在水珠逃离的窗边

2017.6.13
于总行
