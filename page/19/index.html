甩锅文化

发表于 2017-03-04   |   分类于 个人随笔   |  

夜里关了手机，一宿无梦，睡到自然醒，醒来一切都那么安静，好久没有这般舒爽感受了。近来大多数时间都是闹铃将我叫起，噢，又该上班了！过去的这二十天里，由于公司项目赶进度，上了十七天班，只休息了两天，今天是第三个休息日，而且其中大概有十天晚上都加班过了九点，实在是疲惫不堪。回家倒头就睡，上班久坐不起，日子久了，头昏脑涨，并感觉腰部有点疼痛。

确切来说，不是有点儿疼痛，而是年后到现在几乎每晚睡觉都会痛醒。是福是祸，这说不准，爸妈比较担心，说让我有空去医院检查检查。我倒觉得这是个“好毛病”！大概每晚睡到六个小时左右，就会开始疼痛，这样既有了之前平稳的睡眠，又可以早起，而且这毛病也怪，起来一会儿人就好好的了，没有任何不适感，就睡觉时才有，也许是命运之神在提醒我，该早起啦！

哈哈，言归正传，扯了一堆闲话，该说说近日学习到的一种新公司文化了，我将它取名为“甩锅文化”。顾名思义，就是将背在自己身上的锅丢给别人，锅且不论黑白，只要一接到手上，迅速丢出去，自己一身轻松。起初培训时听了不少介绍公司文化的讲座，独独对此“甩锅文化”没有耳闻。为何我近日发觉到了呢？如你所想，我定是有许多亲身体验的。

诗意生活

发表于 2017-03-02   |   分类于 诗歌   |  

早晨一开门
一束阳光打进来
呼，今个儿是个好天！
枝头凉风轻拂
有不知名的鸟儿欢叫

一定有诗意生活的人
在这绿树成荫的小城里
他们有自己的作息
他们享受音乐与沙滩
他们倾听夜空和繁星
他们啜饮咖啡
阅读洒满书籍的阳光

也许未曾谋面
但我确信
小城里，一定有诗意生活的人

巷弄

发表于 2017-03-01   |   分类于 诗歌   |  

沿不太陡的小径往上走
渐渐远离闹市
脱去嘈杂的外衣
脚下便有轻风

旧宅沐浴和暖的阳光
电线胡乱地搭架
蔬菜刚刚入锅
窄窄的小巷幽幽为我带路

曲径狭长望不到尽头
不知其通向何处
唯闻钢琴声星星点点
仿佛溪水静静地流淌

每每迈步向前
儿时老家的记忆就愈发清晰
老家近旁也有窄窄的小巷
黄昏是一支金色的歌曲

画笔用心调配大自然的色彩
远山深浅黑白影影绰绰
羽毛球飞上落满霞光的屋顶
再不愿醒来

日记本

发表于 2017-02-26   |   分类于 诗歌   |  

夜晚读书写作时
书桌上总有你最甜美暖心的微笑
陪伴我陷入深深的迷惘
也陪伴我写下对生命的每一次感悟

翻看日记本
无意间发现
原来你是我诗意的启发者
你将优美的诗与我分享
那潜藏的诗歌的种子
不经意中已在我的心里长出小小的嫩芽

读着曾经写下的我们之间的点点滴滴
回忆浸湿了我的眼眶
那是美的令我思念的感动
往事那么真
一幕幕在字里行间漾开
一笔笔流淌出心灵的无限喜悦与沉醉

你说感谢我记录下这许多美好珍贵的瞬间
可你却不知道，正是你啊
我亲爱的恋人
是你，让我把生活过成了诗

我奢想，我有我们全部的记忆吧？
哦不！我只拥有一半的记忆
还有一半该在你那儿
又或是
我们都拥有全部的记忆
关于我们爱情全部的记忆

这日记是秘密的
这是只属于我们两个人之间的秘密
可我担心，它终有一天会成为公开的秘密
有另外一颗心灵在另外一个时空里
开启这岁月的封条
那时，一定有轻柔美妙的乐曲从匣子里潺潺流出

2017.2.26
于家中

想念慢

发表于 2017-02-26   |   分类于 诗歌   |  

我搅动杯中的咖啡
时间斜过楼顶圆孔状的沙漏

童年的时光渗进糖葫芦
我闲坐在湖边
看太阳的脚步缓缓从眼前走过
草木被晒得瘫软在了风的怀抱

为何周围人们的步伐如此匆匆？
为何轰鸣的发动机再也没有了等待行人的耐心？
为何商人的眼里再没有秋日的一片落叶？
为何我止不住这无休止的思索？

我知道，阳光每一天都会照亮小镇咖啡馆的木质桌椅
而我一生当中
又能有几天，在这儿默坐
品这缕悠闲的光？

我搅动杯中的咖啡
芳香肆虐

哈哈镜

发表于 2017-02-24   |   分类于 诗歌   |  

镜子里的人矮了
大盖帽险些触及豆浆
我将脸蛋埋进小碗
奇怪，我的椅子被谁弄翻了？
我哈哈大笑摔个四脚朝天
却惊奇地发现周围满是
同样的
齿牙未全的大笑

镜子里的人胖了
两张没写完的作业纸趴在桌上沉沉入睡
阳光越过窗沿
梦被揉碎在我的双眼间
脚踏板与上课铃一同涌进教室
寒冷枯枝旁
与我无尽奔跑的
是笔尖下不竭的蓝色墨水

镜子里的人瘦了
玫瑰花如秋风般散落
我的心是一片海
有海鸥和气球在飞翔
海边是宽大的广场
小烘焙坊藏不住咖啡的香气
我的鼻子像一把镰刀
不住裁剪这味道

镜子里的人高了
胡渣洋洋洒洒飘下
我望向自己深邃的眼眸
噢，我的身体哪儿去了！
领带和西服在空气中跳起恰恰
可我分明还感觉到我的存在
哈哈镜笑了
哈哈镜和我一起碎了一地

2017.2.24
于海滨六楼

门外的夏天

发表于 2017-02-18   |   分类于 诗歌   |  

我家门前有条河
不宽不窄
一年四季静静地流淌
日子也一年一年徐徐流淌着

天空明朗
午后的阳光暖烘烘地照耀大地
地面反射的亮光刺得我眯缝起双眼
一副慵懒模样

出门沿河漫步
道路宽阔河水悠悠
脊背略微流汗
夹杂湿气的热风股股从河面上方吹来

弧形右转弯的转角处
有个穿纯白短袖上衣红裙子的女孩
在黄绿色的大大遮阳棚下卖熟透的西瓜
霎时一阵清凉扑面而来

啪嗒啪嗒
拖鞋带着我缓缓挪步摊前
她眼带笑意好像春风，我知道
她的名字叫夏天

2017.2.18
于总行
