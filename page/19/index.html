自然之光

发表于 2017-02-12   |   分类于 诗歌   |  

享受独自一人的周末时光
房间巨大的落地窗贪婪呼吸着光线
我静静感受明暗每秒每分的变化
窗格木影在书页上悄然起舞

多么宁静美妙的光啊
你们形成与存在于多么久远的纪元之前
我要大声赞美耀眼的日光
也愿温柔歌颂皎洁的月光

我深知，我只是你们万千爱慕者中的一个
过去有许许多多赞美过你们的人
将来还会如此

夜色垂压下来
我沉浸在周身弥漫的黑暗里
等候月光照到我的窗前

了无谐

发表于 2017-02-11   |   分类于 诗歌   |  

一盏明灯亮在案头
有如黑暗中指明航向的灯塔

哲人先知们聚会在我的房间
讨论着生命的意义

每本书里都藏着一扇窗
我看到窗外一种种生活的景象

惯性的人生常使我麻木
麻木中我一刀刀捅向自己年轻激情的梦

胡诌八扯的人生观太多
说久了好似他们都懂该怎样实现生命的价值

我是聋子也是哑巴但不是瞎子
让那些肤浅无趣的观念统统都见鬼去

我依然有改变世界的梦想
隐于这首凑够了十四行的诗里

下午茶

发表于 2017-02-09   |   分类于 诗歌   |  

没窗子的屋
下午四点该出去走走
看抱小孩的妇女晒温和的阳光
听对弈老翁间迸发不息的马嘶炮鸣

我带着买下午茶的任务
行走间少了几分惬意与释然
成堆食物从货架朝我的购物袋涌来
仿佛迷途的孩子找到回家的路

迷途的孩子还未坐热屋里的椅子
野兽便群拥而入饱腹而散
我捡起茶水间最后一颗烂了半边的草莓

被他厌弃了大半辈子的残缺救了他一命
就像食人族献祭前突然放走缺了食指的国王
有个疯子在坡顶高喊：“草莓，爹爹在这儿！”

暗潮

发表于 2017-02-07   |   分类于 诗歌   |  

要等的公交刚好过了一辆
一个女生和我
和空空的站台

索性闲下心来
细细打量起这车来人往的街巷
打量起被晨光涤净了夜的欲望的都市

黎明与黑夜
滚烫的暗潮涌动般
欲望在抬头

噢，公交，你来得再慢一点儿吧
我不愿过早进入只能倚靠白炽灯照亮的室内办公
我想再多看看日光白昼下的一切

二十分钟过去
那个女生和我
和挤满了人的站台

终于还是上车
隧道里灯光交替
公交车门上“黄色区域”四个大字交相投影在车内每个人的身上

哦，该死！
低头打字的瞬间我又错过了什么？
这清晨最真实纯粹的厦门

石板凳上的少女凝神呆望
工地旁的大叔仰头张望
而我的目光正望向他们望着的地方

车内音响突然高声放起第九套广播体操
噢，这前胸挨后背的窄小空间
哪容得下乐动欢腾的青春之舞！

我下车一路小跑
在清醒的清晨
为捕捉易在白日脱逃的诗的灵感

啊，滑稽的清晨
我又悄悄将诗意赶走
大步迈进消耗我过剩精力的方格子小屋

2017.2.7
于上班路上

归途

西天耀眼的金光燃烧着大气
喷薄的霞光如流水一般
从浓密的阴云堆里倾泻而下
云的后方
浅浅的蓝白色高空那么辽远
仿佛另一个世界

水城威尼斯的波涛
悠悠荡漾在身后
成群的海鸥飞过头顶
那自在的弧线
三三两两
像涂鸦般在宁静古老的天空下任意勾勒

山岗上错落有致的电线杆
丛林卫士般屹立
岩石边微微摆动的芦苇
晚风中低语
一束焰火孤独绽放在大山深处
那儿正有欢庆的人家

车内笑语不断
车外的灯光被速度拉成了线
驿站里沸沸扬扬
古道外轻蹄踏雪
轮子上是移动的家
家和我一路飞驰
故乡在后方
故乡也在前头

一个小小的哲学故事

发表于 2017-01-23   |   分类于 个人随笔   |  

今晚想起了一个哲学家讲物理的小故事，是我以前在书上看到的，颇感受益无穷，不能不说是“听君一席话，胜读高中物理学”呐！就让我以柏拉图的叙述方式来写吧，主要对话人物是苏格拉底和哆啦小默。故事是这样的：

是日，天朗气清，惠风和畅，城镇里勤劳的人们早早起了床，背上锄头铲子，哼着古老又轻松的旋律下地干活去了。哆啦小默也不例外，刚刚填饱肚子，便阔步走向绿油油的田野了。

哆啦小默是镇上地地道道的农民的儿子，为人憨厚诚恳，对有智慧的读书人很是欣赏，也很喜欢和他们聊天，虽然大字不识几个，却一心向往真理，在邻里间也很受乡亲们的欢迎。苏格拉底觉得哆啦小默是有慧根的，在心里琢磨着启发他一下，于是也来到田埂上踱步晒太阳。

正巧，两人撞个正着。苏格拉底说：“哆啦小默，我听闻街坊邻里称道你的为人，并说你心中向往真理，那么我可以认为你是一个渴求知识的人吗？”

哆啦小默闻言，停下脚步，回答道：“可以，我一直都对知识充满渴望。苏格拉底，听你这么问，想必你是有什么东西可以教导我的吧。”

苏格拉底笑着说：“我是个无知的人，我没有什么可以教导你的地方。不过，我突然想到一个问题，你愿意帮我分析一下吗？”

安静难得

发表于 2017-01-15   |   分类于 个人随笔   |  

今天是2017年的1月15日，从去年7月15日算起，我工作有整整半年时间了。这半年我有了什么样的改变呢？恐怕我自己也说不上来。变得更谙于人际了？也许吧，虽然从前我也觉得自己早已深谙形色人等的心理。变得更加现实了？也许吧，在现实中生活，心态和观念随之变化也是再自然不过的事了。变得更希望安静了？这是唯一我能确定的事了。

如今我觉得安静是十分难得的，甚至可以说是有点儿奢侈的渴望了。平日无论是工作还是生活上的琐事，总缠绕着我，有时真的感觉无法脱身，想有点儿自己的安静的空间好难。

昨天坐在书桌前，听着静谧的钢琴曲翻着书，突然好怀念这种感觉，有多久我没有这样久久地沉浸在这恬静的时光中了？我那被欲望搅扰的不安的心也慢慢平静下来。大起大落、大喜大悲终归是生活中的偶然与调剂，平静地过好每个阳光洒满大地的日子才是最实在与长久的。就像万古长久以来，物种的发展总是缓慢的进程一样，我们也只有在缓慢的闲适时光中，才能生活得更亲近自然与宇宙之法。快感与兴奋终究是闪瞬即逝的，狂热与激情消耗着生命的能量。

每个时代，每个地方，我想，总是不乏嘈杂的人与事的，成天说个没完、大呼小叫、趾高气昂的无知的人，我喜欢戏称他们的人生为“叫嚣人生”或“咆哮人生”。我以为他们的生命是轻的，是浅的，是看不大生命的博大的。在叫嚣之余，他们能在安静中做些什么呢？我不得而知，却也不愿知晓。

一顿晚饭吃得我清醒

发表于 2016-12-27   |   分类于 个人随笔   |  

久未在夜里写博文了，苟非言语在心中满溢，我可能还是睁一只眼闭一只眼淡看社会人情而诗意地在这充满欲望的城市里偷安。可是今晚一顿饭，却吃得我清醒，吃得我把嚼碎的米饭和菜肴酝酿成不吐不快的文字，在此月色皓洁的夜里倾泻出来。

我本一度不喜加班，并视加班为拖沓低效工作的结果。加班是会成为习惯的，是永无天日地驱使自己围着磨恶性循环地绕下去。我所理解的加班，是在手头工作确实很紧急、时间真的不够的情况下，出于事业心和责任感而舍得花费自己更多的时间投入其中以确保工作能够完成的一种自发的行为。加班不是盲目地跟风，不是年底任务多大家都加班自己就一定要跟着加，因为大多数时候，这实在不过是拉长了时间，而完成的工作量却不一定比得上平时高效工作时所能完成的量。加班也不是为面子，好像自己不加就对不起公司，或者对不起谁，我觉得加不加班与内心的愧疚感沾不上半点儿关系，我们真正该想一想的是，我是不是对得起自己付出的时间与生命的热情。在我看来，若按加班时长来定绩效，则这个企业的管理与裁决好不到哪儿去。更有甚者，若强制安排加班，那我想这个企业的员工在大白天里该是多么无所谓与懒散地度过啊，“反正晚上要加班，白天我干嘛拼死拼活地干？”也许便会成为大多数员工的日常心理状态，于是乎，徒劳无功，想来这个企业必定也笼罩在一番末日即临的衰颓之象下了。

人气

发表于 2016-12-13   |   分类于 诗歌   |  

人气是茶水间的水壶咕噜噜冒着的热气
是书架上一排排摆放整齐的书
是沙发上拼好的拼图
是打开又关上的窗
是擦得蹭亮的木地板
是洁净的车前挡风玻璃
是信箱里多了又少了的明信片
是餐桌上没啃完的半块面包
是清晨小巷里充满希望的歌声
是阳台上晒着的蓝白色衬衫
是一夜酣甜的睡眠

人气
是盆里堆积如山的臭袜子
是被子每天变化着的褶皱
是夏天浑身的汗味儿
是客厅忘关的电扇
是厨房除不净的油烟
是键盘边三天没扔的泡面盒
是马桶里残留的一丝余韵
是鞋底粘着的泥
是雨夜屋檐下的一盏灯

我常常怀疑自己是不是真的生活过
唯在这四周
留了些人气

2016.12.13于东区
