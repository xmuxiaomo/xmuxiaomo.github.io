停车场的路灯熄了
笼罩这座城市的夜色逐渐淡去
枝条上叶子瑟瑟发抖
厦门也有了冬的冷意

一条条简讯划过

我们都怀揣同样的罗盘

安静难得

发表于 2017-01-15   |   分类于 个人随笔   |  

今天是2017年的1月15日，从去年7月15日算起，我工作有整整半年时间了。这半年我有了什么样的改变呢？恐怕我自己也说不上来。变得更谙于人际了？也许吧，虽然从前我也觉得自己早已深谙形色人等的心理。变得更加现实了？也许吧，在现实中生活，心态和观念随之变化也是再自然不过的事了。变得更希望安静了？这是唯一我能确定的事了。

如今我觉得安静是十分难得的，甚至可以说是有点儿奢侈的渴望了。平日无论是工作还是生活上的琐事，总缠绕着我，有时真的感觉无法脱身，想有点儿自己的安静的空间好难。

昨天坐在书桌前，听着静谧的钢琴曲翻着书，突然好怀念这种感觉，有多久我没有这样久久地沉浸在这恬静的时光中了？我那被欲望搅扰的不安的心也慢慢平静下来。大起大落、大喜大悲终归是生活中的偶然与调剂，平静地过好每个阳光洒满大地的日子才是最实在与长久的。就像万古长久以来，物种的发展总是缓慢的进程一样，我们也只有在缓慢的闲适时光中，才能生活得更亲近自然与宇宙之法。快感与兴奋终究是闪瞬即逝的，狂热与激情消耗着生命的能量。

每个时代，每个地方，我想，总是不乏嘈杂的人与事的，成天说个没完、大呼小叫、趾高气昂的无知的人，我喜欢戏称他们的人生为“叫嚣人生”或“咆哮人生”。我以为他们的生命是轻的，是浅的，是看不大生命的博大的。在叫嚣之余，他们能在安静中做些什么呢？我不得而知，却也不愿知晓。

一顿晚饭吃得我清醒

发表于 2016-12-27   |   分类于 个人随笔   |  

久未在夜里写博文了，苟非言语在心中满溢，我可能还是睁一只眼闭一只眼淡看社会人情而诗意地在这充满欲望的城市里偷安。可是今晚一顿饭，却吃得我清醒，吃得我把嚼碎的米饭和菜肴酝酿成不吐不快的文字，在此月色皓洁的夜里倾泻出来。

我本一度不喜加班，并视加班为拖沓低效工作的结果。加班是会成为习惯的，是永无天日地驱使自己围着磨恶性循环地绕下去。我所理解的加班，是在手头工作确实很紧急、时间真的不够的情况下，出于事业心和责任感而舍得花费自己更多的时间投入其中以确保工作能够完成的一种自发的行为。加班不是盲目地跟风，不是年底任务多大家都加班自己就一定要跟着加，因为大多数时候，这实在不过是拉长了时间，而完成的工作量却不一定比得上平时高效工作时所能完成的量。加班也不是为面子，好像自己不加就对不起公司，或者对不起谁，我觉得加不加班与内心的愧疚感沾不上半点儿关系，我们真正该想一想的是，我是不是对得起自己付出的时间与生命的热情。在我看来，若按加班时长来定绩效，则这个企业的管理与裁决好不到哪儿去。更有甚者，若强制安排加班，那我想这个企业的员工在大白天里该是多么无所谓与懒散地度过啊，“反正晚上要加班，白天我干嘛拼死拼活地干？”也许便会成为大多数员工的日常心理状态，于是乎，徒劳无功，想来这个企业必定也笼罩在一番末日即临的衰颓之象下了。

人气

发表于 2016-12-13   |   分类于 诗歌   |  

人气是茶水间的水壶咕噜噜冒着的热气
是书架上一排排摆放整齐的书
是沙发上拼好的拼图
是打开又关上的窗
是擦得蹭亮的木地板
是洁净的车前挡风玻璃
是信箱里多了又少了的明信片
是餐桌上没啃完的半块面包
是清晨小巷里充满希望的歌声
是阳台上晒着的蓝白色衬衫
是一夜酣甜的睡眠

人气
是盆里堆积如山的臭袜子
是被子每天变化着的褶皱
是夏天浑身的汗味儿
是客厅忘关的电扇
是厨房除不净的油烟
是键盘边三天没扔的泡面盒
是马桶里残留的一丝余韵
是鞋底粘着的泥
是雨夜屋檐下的一盏灯

我常常怀疑自己是不是真的生活过
唯在这四周
留了些人气

林肯MKC试驾

发表于 2016-12-10   |   分类于 个人随笔   |  

“只是因为在人群中多看了你一眼……”自从今年11月18日在厦大门口第一眼看到林肯MKC起，我就对这款车心心念念。

那天早晨，阳光明媚，一辆红色的前脸酷似盾牌的SUV车迎面而来，缓缓驶过眼前，我看了眼车标，不认识，看到车后面写着“LINCOLN”，知道是林肯。那天白天上百度搜了一下，原来是林肯MKC。不知为啥，这一抹红色点燃了我内心对车的狂热。

近来这段时间，我喜欢上了观察车。开车时看路上的车，工作日午饭后看停车场里的车，周末看世贸楼下的车。法拉利、兰博基尼、宾利、劳斯莱斯、玛莎拉蒂都见到过，保时捷、宝马、奔驰更为常见，林肯还是很少见，一共就看到过五次左右吧。在感慨厦门的豪车和有钱人真多之余，我心里也默默喜欢上林肯这个名车品牌，对车的喜爱让我觉得尘世很美。

古时的人爱马，英雄豪杰都有自己的战马，驰骋四方。如今的人爱车也是一样，车确实是生活中重要的东西。以前关注得不多，觉得这些都是身外之物，可有可无，不明白为什么人们总是在说钱啊、车啊、房啊。现在刚毕业混社会没多久，就渐渐感悟到生活就是无尽地花钱，出门干啥别人都要收你钱，赤裸裸的现实啊，不服不行，没钱寸步难行。是的，寸步难行。房子，是夜里住的地方，安身之处，是睡眠休息的保障。车呢，是白天陪着你出行的伴侣，是外出的代步工具，有车真的是方便、舒心的事，周末偶尔来个自驾，随心所欲、纵情飞驰在大地之上，那也是自由的感觉。

周末白日

发表于 2016-12-04   |   分类于 诗歌   |  

坐在海边看光线
从阴云的间隙穿透而下
演武大桥上车流不息
海风吹起各色裙摆轻轻飘动

红色的兰博基尼和法拉利
留在来往过客的镜头里
白色的保时捷敞篷跑车
倏忽划过一道波浪般优美的弧线

久违了
早晨十点的阳光
久违了
午后甜美的酣睡

周日一杯微微加糖的拿铁
使我暂时忘却了人间还有痛苦
小球潇洒飞越广阔的草场
我又记起了梦想和远方

周末是一剂麻醉针
打得我清醒
可在阳光正好的沙滩上
我却眯缝着眼迟迟不愿醒来
