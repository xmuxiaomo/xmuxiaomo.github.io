致我的爷爷

发表于 2018-03-21   |   分类于 诗歌   |  

山川相连，无远弗届
春花又开放
天空蓝得没有痕迹
干净的土路静悄悄延伸

十三年
时间让悲伤流逝
你远行的日子
你回归的日子

悲伤流转成怀念
怀念流转成为坚定
流转哟
变成了淡淡的温柔的喜悦

我们成为你年轻的日子
每一个你希望的日子

2018.3.21
于北辰山

四点半的下午

发表于 2018-03-09   |   分类于 诗歌   |  

带着远游的心情
重新定义一个下午

列车载着我驶过光影斑驳的大桥
疾驰
拉长易逝的时光

下午四点半
咖啡飘香的步行街
几把闲散的室外座椅
人们戴着茶色眼镜
或者，干脆眯着眼迎向夕阳

树木、楼房
鲜花、笑脸
两辆色彩明丽的豪华小跑
一切都沐浴在金色的柔光里

无意雕琢的山丘
吐纳芬芳的野田
径自流淌的溪水
甘于沉默的岩石

城镇与荒郊交替浮过窗前
一如江天横掠的鸥鹭

2018.3.9
于返家动车上

被爱包围的童年——新春抒怀

发表于 2018-02-22   |   分类于 个人随笔   |  

刚过完春节，又回到熟悉的、几日不见的北辰山。一到山中便遇一场大雨，欢快的雨点落在脚旁，像是欢迎我的归来。中午冲咖啡的间隙，望了望远山和田野，绿油油一片，旺盛的生机丝毫没有被微冷的冬季遮掩。我像看一位许久不见的老友那样看着山野，别来无恙，很是欣喜。

这次回家是在除夕的前一天，请了半天假，提早一点回家与爸妈团聚。家乡年味不减，小小的城市依旧张灯结彩，祥和喜庆。什么是回家？回家就是亲人未见面时彼此的牵肠挂肚，回家就是背包里写满温馨祝福的两个红包，回家就是出了动车站便看见爸妈的身影就在出站口的转角，回家就是自己喜爱的食物总是满满当当地在桌上冒着热气，回家就是放下行李和琐事回归单纯日子的乐趣。

用一日清闲换内心平静

发表于 2018-01-25   |   分类于 个人随笔   |  

平日总感身体倍儿棒，不料近日偶感风寒，加上上班久了，没什么变化，只觉日子过得飞快，很多白昼美好的时光都没有好好珍惜就过去了，于是今个儿请了病假，也算工作日偷得一天安宁。

我和许多人休假的状态也许不同，有的人休假了还时不时要翻看邮箱，时不时查看工作微信群，担心有人找而没及时回复，会失去和错过些什么。我则不然，休假时心安理得，倍感欢愉，对工作上的事儿无牵无挂，工作时认真对待便是，下班或休假便完全隔绝，迅速回到自我最简单自在的生活状态中。这样也算劳逸结合，张弛有道，有的放矢吧。

我喜欢的清闲是什么呢？很简单，早上睡到自然醒，吃一顿美味的午餐，和心爱的人一起觅得一处海边咖啡屋，静静坐着发呆也好，读书也好，说话也好，就这样一直待到繁星满天，然后归去，睡个安稳的大觉！不多吃，不多想，不多关注其他外在的东西。我今天就是这样度过的，清静闲散的一天，找回内心的平静与真实。

失而复得的喜悦

发表于 2018-01-21   |   分类于 个人随笔   |  

最近日子过得比较浮躁，工作日繁琐的社交，心底涌动的欲望，亲戚无休止的谈话，上下班拥堵的翔安隧道，亲人间每日的情绪波动，以及无尽的衣袜和锅碗瓢盆，将岁月的诗意与安宁的时光冲刷殆尽。多浮躁的生活，书桌上竟不时落满尘埃，架子上的书没拆封的依旧没有拆封，细小的打扰可以轻易地中断灵动的思绪和萌生的诗意。

唉，不抱怨太多了，日子本就如此，繁繁复复，谁不是在夹缝和时间的碎片中寻找片刻自由？谁不渴望轻松自在地做自己？谁不想要说走就走的旅行？归根结底，一切都源自我们的比较心，源自无底的欲望，源自不安的灵魂。

日子一匆忙浮躁，诱惑一五光十色，记忆就脱轨。最近时常感觉自己对生活的记忆衰退得太多，常常记不起前两天都做了什么？这个月去了什么地方，几个周末是怎样度过的？今年都干了些什么，回了几次家？自己身上发生了什么重大的事情和难忘的成长？即便在当下用心体会，但是记忆却好像对生活愈加麻木与迟钝，记不住生活中很多值得珍惜的美好的细枝末节了，只有粗线条的，仿若做梦一般的印象。有时甚至分不清一件事情到底是梦中出现过的，还是现实生活里确实存在的。这种感觉真是太糟了！虽然我有时提醒自己要注意觉察，从旁观者的视角看看自己，但这份提醒本身，却泄露了我真正最缺少的、容易忽视的东西——当下的觉察。

当下的觉察少了，生活就容易陷入平庸与瞎忙，每天穿戴整齐出门，好像自己就像个人物，其实，这也许是我自己最鄙视的生活状态，靠外在的东西和无意义的充实来填充自己，这不正显示了自己的空白与无知吗？

小趣

发表于 2018-01-08   |   分类于 诗歌   |  

丙申冬至，大巴返家途中，见金福老师在朋友圈发一水天图，配字：云在青天水在海。于是，即兴对出下句：水戏云影云未睬。觉得有趣，怡然自乐，以为就此作罢，不料金福老师仍不过瘾，说我对得颇具故事性，望能补全四句，成一动漫小剧。于是，哼哼然，嘴里吟哦起来，遂又写下两句：待得天公呼急雨，水落始是故人来。至此圆满。
