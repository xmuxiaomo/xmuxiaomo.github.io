使用与养护

发表于 2022-07-17   |   分类于 诗歌   |  

躺睡十二个钟头
才重建起身体清晰的水平衡
日子彩色便条般贴满时间之墙
疲惫、困倦、焦躁
子时求索，午时不寐
都被重力沉沉沉入无梦之境

不慎碰落的剃须刀
抖出岁月几许
越发断水的钢笔
清洗饮墨新生
齿间美味刻蚀成垢
击碎牙石，口腔洁净

无知觉的自我使用
消耗的，遗忘的
在一个自然醒来的午后
苏醒

内裤，短袖短裤
两只袜子，两只鞋
镜片遇阳光渐变墨色的眼镜
手表，帽子
和无名指上烙进落日余晖的戒指

才够应付工作日一本正经的戏谑
以及夏季冗长白昼的无尽鸣响

成人类文明的遗迹

在郊野阒静的无人区
摘下帽子，扔掉手表
抖落身上十一片绿叶
沿破败矮墙裸跑
溶进树影黄昏

观云小感

发表于 2022-06-09   |   分类于 自然观察   |  

掐指一算，加入总部在英国的赏云协会（Cloud Appreciation Society）已半年有余，平日久坐的脖子僵硬和肩颈酸痛得以缓解，因为常常抬头看天，左右前后到处转。记得林夕开玩笑说，“时常看星星可以治好颈椎病！”看云一定要注意护眼，白天天空光线很强，最好戴个防紫外线的墨镜或寻找附近的树木、路灯、屋檐、墙等遮挡物，实在不行就伸手挡挡阳光，避免直视太阳而损伤眼睛。前不久我因每日在山间强光当头的正午看云太久，眼球上出现不少血丝，而且晚上易困，不想再看别的东西，只想睡觉休息，应该就是强光对眼睛造成了一些损伤吧。

这半年的观云实践带给我无限的惊喜和激情。观云实践，我喜欢这么说，因为看云不能光从书本理论上认识云的形成原因和天气条件，还要花大量时间观云，欣赏天空，通过实践来认识天上的各种云彩和光学现象，以及某种云的出现和周围天气的联系。回想起来，我爱上看云，源自小说《岛屿之书》，读到奥克尼群岛仲夏时节持续三周的极昼，午夜的天空偶然会出现“夜光云”。出于好奇，我搜了搜“夜光云”，接着是一系列延伸的网上冲浪：乳状云、贝母云、落幡洞云、荚状云、山帽云……太神奇了！于是一发不可收拾，整日沉迷云中。后来买了赏云协会官方出版物《云彩收集者手册》（The Cloud Collector’s Handbook，由英国的加文·普雷特-平尼著），注册了会员（会员编号：57319），手机下载了App “CloudSpotter”，邮箱每日会收到一张精美的云朵图片和一段诗意的文字说明。就这样，我开始了赏云之旅，手册与App相得益彰，中英文介绍了四十多种云彩和大气光学现象，从开始只对几种特别罕见的云（如荚状云、夜光云、开尔文-亥姆霍兹波、糙面云、马蹄云等）感兴趣，到后来发觉每种云有其独特的美，越了解便越喜爱。每当看到刚刚习得的新云种出现天空，我便立马欣喜地打开App拍下并提交给协会评审，评审通过后集云分和徽章会不断增加，全球排行不断靠前，相当有乐趣和成就感。目前我的集云分83/115、徽章11/20，全球排行483，已经成功收集了31种云和光学现象。

论勇气与我的生死观

发表于 2022-04-17   |   分类于 个人随笔   |  

今天，第一次带小女儿莎莎逛动物园，在集美的灵玲国际马戏城旁。莎莎再过七天就一周岁两个月啦，最近在学走路，已会咿咿呀呀说话了，小小的脑袋，知道的不少，不仅会认得亲人，家里的电器、日常用品也能记住，还会模仿挺多声音。

以前去马戏团看过表演，欢乐的气氛带给过我快乐。但后来我和梦梦都觉得不忍心再去看此类动物表演，因为训练太残酷，动物很可怜。虽说关在动物园里的动物也失去自由，因人之间的交易而受到伤害，但偶尔与家人、孩子一起看看，亲近动物，认识动物，算是私心下的一种心理妥协吧，毕竟我们大多数人少有野外生存方面的练习，很难真的到纯粹的大自然里去与动物共同生活，近距离观察它们。小朋友与动物是很容易亲近的，他们纯真无邪，自己也是小动物，没有观念，内心纯净，可以原始的方式与动物交流，这点很珍贵，我们应永远如孩子般至简、至真啊。

莎莎刚进动物园那会儿，坐在小推车里睡着了，母亲在前头走，与姐姐的两个小龙凤嘟比、嘟米一起玩，鹦鹉叽叽喳喳，我在后面慢慢推着推车走，陷入一种强烈的情绪当中，这情绪包裹了我一整天。可能是以前看到过这类报道，我在想，如果至为亲近的人不小心落入虎群间，我会怎样？随即便有答案，我将毫不犹豫跟着跳进去，与亲人并肩而战，不论结果。如果犹豫不决或因任何迟疑而胆怯后退，我想我必定会终生抱憾，无法忍受目睹亲人无助、恐惧，在不知所措中面对强敌而死去。落入虎群的含义可以衍生，狮群、豹群、鳄鱼群，恐龙，外星生物，急流旋涡，无边大海，风暴雪崩，人为约定的令人目眩的律法、审判，无法用语言沟通的战争中的异国士兵，或者坠入星空和无尽黑暗的宇宙深渊。

发表于 2021-10-09   |   分类于 个人随笔   |  

总爱和人开玩笑，又或许认真，说我觉得什么时候应该辞职，辞在灵感枯竭时。判断灵感是否枯竭，很容易，隔一段时间回头看看，成诗几首？是否已到山穷水尽，无词可依、无话可说的境地？

从前我以为，当工作过多地侵蚀生活，过多地占据私人时间，工作得越来越认真投入的时候，便可考虑辞职了，因为工作不过是谋生的手段，是生活可以持续的方式，而非我真正的追求所在，不值得花这许多时间，浪掷青春。但如今我感到，这个因素区居第二。是不会享受闲暇时光了吗？是生活毫无羞耻的重复使我麻木了吗？是荷尔蒙总在深夜勒紧我的脖子让我无心静省吗？是三十岁上有老下有小的日子重压在我肩上吗？是工作定期奖赏的蝇头小利蒙蔽了我的视线吗？是日常人际交往的诸多情绪索萦心间吗？这些也许值得反思，但只要我的灵感还未枯竭，仅有的不多的高光时刻偶尔降临，我便满心欢喜。

闯北辰

发表于 2021-07-25   |   分类于 诗歌   |  

信步登临北辰
初访一山，犹如探险
古树投下荫凉
在这盛夏七月

周围充满不知名的动植
能信赖的唯有双脚
一步一步
循先人足影

穿行小径
转过木栏
忽闻水声切切
一细流自石坡奔落

我沿着水声
寻此生命之源
夕阳的光辉铺满石阶
古意盈盈

双膝渐颤，喘息愈急
汗如雨下
意识消褪，身躯硕大
我从智人走向了原人！

行至高台，忍不住放声
清冽的呐喊顺溪流逐远
呼唤记忆的山
呼唤不远的曾经

独步山林
趁光线尚明
漫长山路需要休憩的凉亭
正如生活需要旅行

脑中又起乱绪
幸逢一巨蜂
嗡嗡绕耳
将我赶下山林！

（一）
远离非必需的电子设备
沉沉入梦
梦里我清醒独行
脑中吟唱着四句诗行
黑暗中企图抓纸笔记下
恍惚已至海平线切分日出的蛋黄
绝美的诗句消褪
当意识来临

（二）
洗漱，上卫生间，吃饭
每日清晨永恒的调调
婴儿兴奋地躺着，手舞足蹈
新的一天多么欢乐

字句简洁，故事生动毫不拖沓
马尔克斯的叙述总有新意
接水，换纸尿布
婴儿天真无牙地笑

（三）
午后去健身房
空气烤过似的暖烘烘的
街道整洁干净
行人车辆稀少
绿树勾勒出城市的线条
平日里隔着车窗觉不出的美
很热却乐意再多逛几圈
家附近变换不定的橱窗与小铺

（四）
喝水，洗澡，吃饭
配部电影才够周末的味道
又是抗战片
从小到大枪炮声不绝于耳
铭记过去究竟有多重要？
人们总是如此，徒劳地对抗遗忘
忽视眼前生活细碎的美妙
重复仇恨、杀戮，人类古老的敌意

（五）
傍晚小睡
抹去一天里生成的无端思绪
醒来，如初生婴儿般简单
放水，给小女儿洗澡

洗脸，上卫生间，吃饭
看点无聊综艺节目
过惯了荧幕生活的明星
如政客，张嘴却没有声音

（六）
咖啡杯中一圈圈下降的
时间的刻度
口里充满卢旺达
阳光的香气

李斯特衣衫褴褛
哀伤贡多拉沉重地负载着生命
时光如水般流逝
夜晚的歌声响起

“记住，你会死的”

发表于 2021-03-19   |   分类于 个人随笔   |  

上班许久，观察到一个有趣的现象，大多数同事只限于邮件、通讯工具上的往来，真人即使碰面却也不相识，哈哈，名字都是这么些名字，名字之间常来常往，但名字却悬浮在虚拟网络上，远离了名字下方面孔混淆的人群。如今的大公司，组织架构庞大，机构分散各地，人员众多，且不说人员流动大，来的来去的去，光是常驻的老员工，如果部门和上班地点不同，都很难混个脸熟，更不用说有进一步的交往。

而且如今的上班族乃至社会大众，普遍存在注意力缺失、分散化现象。信息时代，尤其是网络和手机出现后，周边事物都在加速发展，一个再重大的消息，都没有了以往长久的影响力，人们看到消息的一刻也许感到震惊，议论纷纷，但用不了多少时辰，他们的注意力就会被新的事物占据，进而遗忘了上一件事，而遗忘仍在不断滋生，一个接着一个。我们很容易就忘记昨天的某个时刻做了什么，发生过什么，头脑中想过什么问题。所以不用担心早晨出门时脸上长了痘、嘴唇上火起泡、彻夜忧伤哭肿了眼，没人会在意，除了你自己，除了很要好的朋友外，周围的人们并不会真的关心你、在意你，他们没有多余的注意力分给你，他们的双眼紧盯手中的一小块发光屏幕，疯狂猎奇，八卦娱乐的，政治新闻的，都一个样，或只醉心于他们所感兴趣之事，沉迷其中，进而自身也成为屏幕，满大街行走着的屏幕，屏幕充斥着拥挤的交通工具，公交车、地铁、走廊、餐馆、下水道，都被荧光发亮的屏幕填满。
