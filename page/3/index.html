正午，如夜

发表于 2021-10-11   |   分类于 诗歌   |  

天空中
亮白色云朵迅速飘过
风指示方向
在远方山顶聚集
云朵也赶集吗？

正午阳光
投下万物的影子
刨去艳丽芳华的外表
留下本质
留下黑白

一群群骨架在大地绘图
静止的是僵死的建筑原本空洞的结构
动着的是草木的枝干
黑色的叶片、花朵
以及现代人类的骷髅之舞

这大地的黑白
像无声电影
一刻不停在上演
生的世界、死的世界
地平线上共存

表象总是变化，易逝
但本质却延续
大地上的影子不区分时空
事件、样貌、工种、言语、情绪
都荡然无存

唯有静默的黑白永恒
骨架移动，停留
黑色的叶片、花朵
以及黑白如夜的正午
璀璨喧嚣的骷髅之舞

扯

发表于 2021-10-09   |   分类于 个人随笔   |  

总爱和人开玩笑，又或许认真，说我觉得什么时候应该辞职，辞在灵感枯竭时。判断灵感是否枯竭，很容易，隔一段时间回头看看，成诗几首？是否已到山穷水尽，无词可依、无话可说的境地？

从前我以为，当工作过多地侵蚀生活，过多地占据私人时间，工作得越来越认真投入的时候，便可考虑辞职了，因为工作不过是谋生的手段，是生活可以持续的方式，而非我真正的追求所在，不值得花这许多时间，浪掷青春。但如今我感到，这个因素区居第二。是不会享受闲暇时光了吗？是生活毫无羞耻的重复使我麻木了吗？是荷尔蒙总在深夜勒紧我的脖子让我无心静省吗？是三十岁上有老下有小的日子重压在我肩上吗？是工作定期奖赏的蝇头小利蒙蔽了我的视线吗？是日常人际交往的诸多情绪索萦心间吗？这些也许值得反思，但只要我的灵感还未枯竭，仅有的不多的高光时刻偶尔降临，我便满心欢喜。

止境，适境

发表于 2021-10-08   |   分类于 诗歌   |  

我在此山中打井
你在别处打
虽周遭风景迥异
然同样用工
地底水源终将我们连通
正如夜里仰望天空
明月的美映在不同地域人们的眼中

我没有语言天赋
英语学习多年
仍止于纸面的读懂
若赏英文诗
很难由一行行字母联想万物
而简体中文
时常品着便神游物外

也如自学音乐
记得五线谱上音符对应的琴键
却难望谱而生乐音
耳畔浮不起旋律
唯看得音符纷纷跃然纸上
肆意飞舞
离我而去

时光在风起的书页间流走
大师大作浩如烟海
年逾三十
更钟一隅之静
一瓢之饮
一言之美
享心中所乐

闯北辰

发表于 2021-07-25   |   分类于 诗歌   |  

信步登临北辰
初访一山，犹如探险
古树投下荫凉
在这盛夏七月

周围充满不知名的动植
能信赖的唯有双脚
一步一步
循先人足影

穿行小径
转过木栏
忽闻水声切切
一细流自石坡奔落

我沿着水声
寻此生命之源
夕阳的光辉铺满石阶
古意盈盈

双膝渐颤，喘息愈急
汗如雨下
意识消褪，身躯硕大
我从智人走向了原人！

行至高台，忍不住放声
清冽的呐喊顺溪流逐远
呼唤记忆的山
呼唤不远的曾经

独步山林
趁光线尚明
漫长山路需要休憩的凉亭
正如生活需要旅行

脑中又起乱绪
幸逢一巨蜂
嗡嗡绕耳
将我赶下山林！

大量与几个

发表于 2021-07-18   |   分类于 诗歌   |  

每一天，有五万辆车
从我家窗前经过
除了偶尔几声突兀的轰鸣
其余都混入时光的白噪音

城市里有三百万
相似而迥异的面孔
除了常见的几百张能够分辨
其余将擦身，永远模糊

街道旁有许多小店
变换着招牌与装潢
几易其主
成为记忆路标的屈指可数

大量无意义的时光堆积
才偶有高光时刻啊
高潮一瞬，迸发
而后复归平淡

重复之事
机械反应
耗散白昼所有的光阴
张驰间，才涌入夜晚无尽的遐思

可能与实在

发表于 2021-07-07   |   分类于 诗歌   |  

我们相遇，说来偶然
茫茫人海于万一
时空恰巧交逢
偌大宇宙
无尽绵延的时光
然后，佚散
散作瑰丽华彩的自由

婴儿出生，想也偶然
生命的起点本就如谜
某个良宵，地点随机
亿万跑者中胜出
无数粒子与元素聚合
于是，你诞生
不着名姓

可一切并不是这样！
新生的婴儿啊，你是元初的质
你是确定的无法再分的本原
你是唯一的真实
没有概率和随机性之说
没有偶然
所有可能汇聚为确信无疑的你
别无二者

或许在意识形成之前，
才最纯粹？
然后分化出无数可能的你？
就像从根发散成枝条
上面挂满成熟的果实——
画家、醉汉、诗人
而你只能长成实在的一个果实
没有平行人生
又或者，你本就集合了一切多样性？

一场大雨
会淹没一首夜晚湖畔的诗
但也可能因此而生
居家闲思的诗
人生单行
可能性的平行世界
想象里存念
思维游戏

身边的风景

发表于 2021-07-06   |   分类于 诗歌   |  

久未步行至筼筜湖畔
幽灵船般漆黑巨大的条形水面
在晚风里黝黝鼓动
载着模糊的灯影
无声流淌

桐城的彩虹桥
塞纳河上的米拉波桥
夜色调画着想象
加上视力不佳
一段路便像走过了世界

天空隐隐透出藏蓝
遥遥悬着几颗星子
光线明明已经很暗
足以将万物掷入黑白
为何摆动的枝叶间
还缓落绿意？

鞋子与轮胎遍布路面
堆放了几个世纪
都不留痕迹
只有吹去又吹来的沙
在大地涂抹
以及留下不幸被命运碾碎的生命

七月一个普通的星期天

发表于 2021-07-04   |   分类于 诗歌   |  

（一）
远离非必需的电子设备
沉沉入梦
梦里我清醒独行
脑中吟唱着四句诗行
黑暗中企图抓纸笔记下
恍惚已至海平线切分日出的蛋黄
绝美的诗句消褪
当意识来临

（二）
洗漱，上卫生间，吃饭
每日清晨永恒的调调
婴儿兴奋地躺着，手舞足蹈
新的一天多么欢乐

字句简洁，故事生动毫不拖沓
马尔克斯的叙述总有新意
接水，换纸尿布
婴儿天真无牙地笑

（三）
午后去健身房
空气烤过似的暖烘烘的
街道整洁干净
行人车辆稀少
绿树勾勒出城市的线条
平日里隔着车窗觉不出的美
很热却乐意再多逛几圈
家附近变换不定的橱窗与小铺

（四）
喝水，洗澡，吃饭
配部电影才够周末的味道
又是抗战片
从小到大枪炮声不绝于耳
铭记过去究竟有多重要？
人们总是如此，徒劳地对抗遗忘
忽视眼前生活细碎的美妙
重复仇恨、杀戮，人类古老的敌意

（五）
傍晚小睡
抹去一天里生成的无端思绪
醒来，如初生婴儿般简单
放水，给小女儿洗澡

洗脸，上卫生间，吃饭
看点无聊综艺节目
过惯了荧幕生活的明星
如政客，张嘴却没有声音

（六）
咖啡杯中一圈圈下降的
时间的刻度
口里充满卢旺达
阳光的香气

李斯特衣衫褴褛
哀伤贡多拉沉重地负载着生命
时光如水般流逝
夜晚的歌声响起

新意与轻松

发表于 2021-06-30   |   分类于 诗歌   |  

从下往上
由迅疾到缓慢
橙色渐变为淡蓝
混几抹霞光
没入星辰黑曜

在每日都走的路上
侧过头，如草食动物静观
调整座椅角度
换几块承受重力的肌肉
用耳机代替外放

视线的倾斜
表盘刻度的偏离
单调进程中的一点点变动
再把幻想涂写进现实
重走的路便像漫游世界

尖利的电钻
企图钻入我白色的梦境
屋外是大片草场和森林
绿意盈窗
轻的风吹过
