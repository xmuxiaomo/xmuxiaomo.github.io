阳台遐想

发表于 2020-01-13   |   分类于 诗歌   |  

白墙似雪
空气中飘来西湖醋鱼的味道
鸟儿爬上青绿色的瓦片
瓦片在我伸手可及的地方

远处的山峰层层叠叠
雾霭幻它们作动态的水墨
阳光盖着寒风里的树丛
和我满头浓黑的短发

民宿顶层的露天阳台
时光安静流淌
没有喧嚣的高地啊
闲静的天堂

只是莫奈的阳光终抵不过寒风侵袭
我得钻过矮门回屋喝杯热水！

2020.1.13
于杭州西湖边乐墅民宿

无题

发表于 2020-01-10   |   分类于 诗歌   |  

月亮躺在枝杈交错的床上
明亮又宁静
脚下的石路与夜空
都蒙上灰白的光

颜色是什么？
在树的眼中，不同生物眼中
光谱迷离，画家捕捉
调色盘单调而匮乏的抽象

干裂的树皮、结疤
贫困的枝丫
腐坏树桩的年轮记录着诞生与疾苦
幸福、孤独和死亡

无可复制的树啊
不语的智者
你独一无二永不重来
你的伤痕、结疤、疾病、新生
你风中不变的沉默

2020.1.10
于庄汉水楼旁树下

发表于 2019-07-06   |   分类于 个人随笔   |  

没有足够的闲暇与独处时光，便难静下心来写作，在夹缝里拼凑与利用碎片化的时间，固然也能够积累与不断学习，但始终是难成体系的。二十多岁呐，多么宝贵又容易蹉跎的年华，二十多岁，多么迷茫又彷徨的年华。

今日得闲，在家中独自消磨时光，泡咖啡时偶得一想法，如果东西不好好加以使用，即便当下是美物，经年累月也终将黯淡无光，而且总感觉物本归物，并不真的是自己所有的。

比如前段时间托4s店买的911 turbo S Exclusive Series限量版马克杯，Made in Germany，很漂亮的杯子。买来后只有少数几次用来喝水，其余时间便闲置桌上，当一摆设。而日常喝咖啡、泡牛奶、喝水都仍用原来自己常用的那个杯子。今日想想，这杯子虽然干干净净，用来喝水也挺好，但是实在没有很深的感情，因为如果都不用它喝咖啡、喝牛奶，不与自己日常所好联系起来，好像就不真的是自己的杯子，而是有一定距离的、“可远观不可亵玩”之物了。哈哈，伴随时日、杯壁泛黄的杯子也许才令我回味，才涂抹上一层共同经历岁月的情谊。

#岁月 #车

钢琴独奏会

发表于 2018-09-28   |   分类于 个人随笔   |  

今晚是张玎苑博士的钢琴独奏会，她弹奏德彪西的作品，以纪念德彪西逝世100周年，今年在音乐界里也被称作“德彪西年”，当音乐厅里琴声响起，世界突然安静无比，没有平日的喧嚣与嘈杂。听现场钢琴声，真的与通过电子渠道收听有很大区别，琴声在音乐厅布满褶皱的墙面上漫反射开，给人一种空灵的感觉，所有的语言都显得累赘与多余，只需静静聆听。

听音乐可以使人慢下来，注意力更好地集中，这与我们平时注意力被极度分散化与碎片化的经验是完全相反的。我坐在音乐厅里，什么都不做，什么都不想，只是聆听，平心静气地让美妙的乐声在耳道里来去。

说到底，生活经验最重要的就是修心。听音乐是修心，开车也是修心，工作中版本管理也是修心，一旦心平气和、平心静气，乐声便会轻易唤醒疲惫的灵魂，道路便开阔与畅达，工作便顺心自如，更无许多不必要的所谓烦恼。

现在已过零点，很困，不多写了。充足的睡眠也是开心生活的保证。不论世界多么浮躁繁杂，重要的时刻都是轻易便以寻常方式降临，打起精神来体验每个不可重来的生命时刻，平心静气地过每个独一无二的日子。

晚安。

Macan S，激情澎湃

发表于 2018-08-14   |   分类于 个人随笔   |  

上周五喜提18款保时捷Macan S，人生的第一辆车，欣喜若狂！

之前和4S店销售杰哥约好提车时间在周五，2018年8月10日，这天正好是我拿到驾照3周年的纪念日。提车那天早上，我穿戴整齐出门，衬衫、西裤、“老爷车”。同事感到奇怪，为什么到周五了，本该是休闲地上上班准备过周末了，我反而穿成这样在山里上班。哈哈，我嘴里说着“穿整齐好好上班”，可内心暗暗的激动情绪当然是外人无法理解的。时间越接近傍晚，我内心的情绪起伏便越大，我知道这也是我人生的一个重大时刻，3月3日订车后等待了五个多月的车子终于要开出4S店，开回家了。

北辰山的夏天很热，虽然时令已过立秋，可白天仍高温不下，尤其到了正午，吃完午饭，便常感昏昏沉沉。以往劳动人民本来吃饱饭是越加有劲、生龙活虎的，可是现在的人不知为什么，也许是长期在空调房里对着电脑干活，身体处于亚健康状态，每当过午吃饱，便越加昏沉，倒头大睡。我平时是颇喜欢在午饭后稍事休息，便开始点灯阅读，泡上一杯可口的咖啡，读点诗歌，读点散文，偶尔眺望远山作诗一首，再惬意不过了。天凉的时候我基本不午休，每天中午都很有精神，伴着周遭此起彼伏的呼噜声，独自享受安静的午间阅读时光。不过近来正值厦门高温天气，午后有时实在困乏难当，便也加入呼噜大军，吃饱便睡。还别说，有时睡醒了感觉的确不错，一个下午都很清醒舒适。

趣谈画饼

发表于 2018-06-13   |   分类于 个人随笔   |  

时常与朋友打趣，说自己在一家画饼公司工作，啥是画饼公司呢？就是公司不断给你注入新的希望，领导承诺公司将给你你所想要的一切，不断有同事传来小道消息说今年的年终奖会比以往高很多！香喷喷的大饼时而如暴风骤雨般打在我们向往美好生活的脸上，时而又如绕着磨转圈的驴子面前挂着的胡萝卜，永远可望而不可及。

生活中总有许多大饼画在比我们跳起来还高一点远一点的地方，像夜里引航的灯塔微弱的光芒，又像沙漠里不时显现的湖泊。虽然大饼有时虚无缥缈，我们也都知道不是光靠想想就能够得到的，但它却可以引导我们不断对抗生活冗长的无聊，使我们不断有继续生活下去的憧憬和期待。这份期待很美，心之所向，持之以恒，像倦怠时一杯暖心的咖啡，为干渴的身体注入新的活力。

你有过这样的体会吗？在得到一件非常渴望的东西之前，人也好，物也好，某种追求的高度和境界也罢，你面对的是漫长而无尽的等待。渴望的事物若即若离，忽远忽近，让你患得患失，甚至觉得自己都快为此得焦虑症或患上精神病了。但这漫长的等待在回过头来看时，加上一勺记忆的美化效果，往往就是甜美的，是值得的，是可以长久回味的，因为这个等待的过程中内心是被某种美好的情感填满的，很充实。生命中不正是需要这种内心充实的感觉吗？
