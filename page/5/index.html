却无法通过树木、石墙

噢，也只是一小部分而已！

闯北辰

发表于 2021-07-25   |   分类于 诗歌   |  

信步登临北辰
初访一山，犹如探险
古树投下荫凉
在这盛夏七月

周围充满不知名的动植
能信赖的唯有双脚
一步一步
循先人足影

穿行小径
转过木栏
忽闻水声切切
一细流自石坡奔落

我沿着水声
寻此生命之源
夕阳的光辉铺满石阶
古意盈盈

双膝渐颤，喘息愈急
汗如雨下
意识消褪，身躯硕大
我从智人走向了原人！

行至高台，忍不住放声
清冽的呐喊顺溪流逐远
呼唤记忆的山
呼唤不远的曾经

独步山林
趁光线尚明
漫长山路需要休憩的凉亭
正如生活需要旅行

脑中又起乱绪
幸逢一巨蜂
嗡嗡绕耳
将我赶下山林！

大量与几个

发表于 2021-07-18   |   分类于 诗歌   |  

每一天，有五万辆车
从我家窗前经过
除了偶尔几声突兀的轰鸣
其余都混入时光的白噪音

城市里有三百万
相似而迥异的面孔
除了常见的几百张能够分辨
其余将擦身，永远模糊

街道旁有许多小店
变换着招牌与装潢
几易其主
成为记忆路标的屈指可数

大量无意义的时光堆积
才偶有高光时刻啊
高潮一瞬，迸发
而后复归平淡

重复之事
机械反应
耗散白昼所有的光阴
张驰间，才涌入夜晚无尽的遐思

#诗

可能与实在

发表于 2021-07-07   |   分类于 诗歌   |  

我们相遇，说来偶然
茫茫人海于万一
时空恰巧交逢
偌大宇宙
无尽绵延的时光
然后，佚散
散作瑰丽华彩的自由

婴儿出生，想也偶然
生命的起点本就如谜
某个良宵，地点随机
亿万跑者中胜出
无数粒子与元素聚合
于是，你诞生
不着名姓

可一切并不是这样！
新生的婴儿啊，你是元初的质
你是确定的无法再分的本原
你是唯一的真实
没有概率和随机性之说
没有偶然
所有可能汇聚为确信无疑的你
别无二者

或许在意识形成之前，
才最纯粹？
然后分化出无数可能的你？
就像从根发散成枝条
上面挂满成熟的果实——
画家、醉汉、诗人
而你只能长成实在的一个果实
没有平行人生
又或者，你本就集合了一切多样性？

一场大雨
会淹没一首夜晚湖畔的诗
但也可能因此而生
居家闲思的诗
人生单行
可能性的平行世界
想象里存念
思维游戏

久未步行至筼筜湖畔
幽灵船般漆黑巨大的条形水面
在晚风里黝黝鼓动
载着模糊的灯影
无声流淌

桐城的彩虹桥
塞纳河上的米拉波桥
夜色调画着想象
加上视力不佳
一段路便像走过了世界

天空隐隐透出藏蓝
遥遥悬着几颗星子
光线明明已经很暗
足以将万物掷入黑白
为何摆动的枝叶间
还缓落绿意？

鞋子与轮胎遍布路面
堆放了几个世纪
都不留痕迹
只有吹去又吹来的沙
在大地涂抹
以及留下不幸被命运碾碎的生命

七月一个普通的星期天

发表于 2021-07-04   |   分类于 诗歌   |  

（一）
远离非必需的电子设备
沉沉入梦
梦里我清醒独行
脑中吟唱着四句诗行
黑暗中企图抓纸笔记下
恍惚已至海平线切分日出的蛋黄
绝美的诗句消褪
当意识来临

（二）
洗漱，上卫生间，吃饭
每日清晨永恒的调调
婴儿兴奋地躺着，手舞足蹈
新的一天多么欢乐

字句简洁，故事生动毫不拖沓
马尔克斯的叙述总有新意
接水，换纸尿布
婴儿天真无牙地笑

（三）
午后去健身房
空气烤过似的暖烘烘的
街道整洁干净
行人车辆稀少
绿树勾勒出城市的线条
平日里隔着车窗觉不出的美
很热却乐意再多逛几圈
家附近变换不定的橱窗与小铺

（四）
喝水，洗澡，吃饭
配部电影才够周末的味道
又是抗战片
从小到大枪炮声不绝于耳
铭记过去究竟有多重要？
人们总是如此，徒劳地对抗遗忘
忽视眼前生活细碎的美妙
重复仇恨、杀戮，人类古老的敌意

（五）
傍晚小睡
抹去一天里生成的无端思绪
醒来，如初生婴儿般简单
放水，给小女儿洗澡

洗脸，上卫生间，吃饭
看点无聊综艺节目
过惯了荧幕生活的明星
如政客，张嘴却没有声音

（六）
咖啡杯中一圈圈下降的
时间的刻度
口里充满卢旺达
阳光的香气

李斯特衣衫褴褛
哀伤贡多拉沉重地负载着生命
时光如水般流逝
夜晚的歌声响起

还是我！

“记住，你会死的”

发表于 2021-03-19   |   分类于 个人随笔   |  

上班许久，观察到一个有趣的现象，大多数同事只限于邮件、通讯工具上的往来，真人即使碰面却也不相识，哈哈，名字都是这么些名字，名字之间常来常往，但名字却悬浮在虚拟网络上，远离了名字下方面孔混淆的人群。如今的大公司，组织架构庞大，机构分散各地，人员众多，且不说人员流动大，来的来去的去，光是常驻的老员工，如果部门和上班地点不同，都很难混个脸熟，更不用说有进一步的交往。

而且如今的上班族乃至社会大众，普遍存在注意力缺失、分散化现象。信息时代，尤其是网络和手机出现后，周边事物都在加速发展，一个再重大的消息，都没有了以往长久的影响力，人们看到消息的一刻也许感到震惊，议论纷纷，但用不了多少时辰，他们的注意力就会被新的事物占据，进而遗忘了上一件事，而遗忘仍在不断滋生，一个接着一个。我们很容易就忘记昨天的某个时刻做了什么，发生过什么，头脑中想过什么问题。所以不用担心早晨出门时脸上长了痘、嘴唇上火起泡、彻夜忧伤哭肿了眼，没人会在意，除了你自己，除了很要好的朋友外，周围的人们并不会真的关心你、在意你，他们没有多余的注意力分给你，他们的双眼紧盯手中的一小块发光屏幕，疯狂猎奇，八卦娱乐的，政治新闻的，都一个样，或只醉心于他们所感兴趣之事，沉迷其中，进而自身也成为屏幕，满大街行走着的屏幕，屏幕充斥着拥挤的交通工具，公交车、地铁、走廊、餐馆、下水道，都被荧光发亮的屏幕填满。

今天是在月子中心度过的第三个周末啦，窗外是下午四五点钟明媚的阳光，浅浅的鸟鸣，一丝微风吹来，午觉后精神很好，咖啡格外美味。泳池碧波漾漾，安静的感觉，让我想到疗养院，不知是否此番情境？
