时间的错觉

发表于 2017-02-14   |   分类于 诗歌   |  

我们相识之初
时间流淌得很随意
一群同学围坐在偌大的教室里听讲
我们偶尔交流
却并不刻意
铃声响起
转身那样轻易

后来
时间变得很悠长
你托着腮凝望黑板
我的目光却不自觉地瞥向有你的窗前
窗外树影斑驳
我呆呆地盯着你手中的笔出神
又常悄悄移动视线
望向你的侧脸

牵你的手
在月色如火般静静翻涌的夜晚
时间仓促地退到地平线以外
我乱了方向
足迹一遍遍将芙蓉湖圈起
寒风吹过脸庞
你手心的温度那样炽热

再后来
时间变得很慢很慢
慢到一日便盛开出一朵花
慢到细沫从浪尖高高腾起
慢到每一秒都像永恒
慢到每一刻都芬芳馥郁
我将每一天
如花瓣般藏进书页

有你的日子
时间变魔术似的忽快忽慢
快时一吻天便老去
慢时我就陪在你身旁
从早晨到黄昏
从郊野到书房
澄净的天空中有米色的阳光

如今
时间从层叠的叶缝间滑过
打碎一地光影
老旧街区的十字路口
我举目四望
越过时光的轨迹
我看到
满满都是暖心的故事

2017.2.14
于老街树荫下

夜逢千杯少

发表于 2017-02-13   |   分类于 个人随笔   |  

零点二十分，睡意全无，我知道即使此刻关了灯躺在床上，自己也久久不会入睡，因为大脑很活跃，浑身的细胞也都苏醒着。

本想蒙头睡去，明早四时起来再写这篇文字，但是兴之所至，情绪到了，无法不写点儿什么。怕明早起时，昏昏沉沉，是啊，过了这阵子，也许再难提笔写下此刻的生命历程与感受。我很珍惜和享受这种意欲书写的快感，愿自己常能有此“踏破铁鞋无觅处，得来全不费工夫”的随性之感。这种感觉是妙手偶得的，是可遇而不可求的，是来临时欣喜而毫不抗拒的，是充斥全身而又十分微妙的。

傍晚下班，在“子母粉丝”店匆匆吃了碗牛肉粉丝，手机屏幕上弹出一条消息——“厦航MFxxxx航班将提前二十分钟抵达高崎机场”，赶忙扒两口粉丝，喝两勺汤，起身便出发去机场。今晚梦梦回来啦，随机的还有她的两位老师。送他们都回到了住处，我便安心赴友人Y之约，到家楼下的“每天咖啡”门口等候。

Y很守时，两分钟后便出现在眼前，我们有一年半多没见面，但却一眼就认出了彼此。

流水

发表于 2017-02-12   |   分类于 诗歌   |  

不尽的湖水在晚风中
汩汩向前流淌
这水行走过的世界一定比我更辽远
我猜不出它流经了多少地方
流走了多少个世纪

身旁的流水流过多少人的身旁
它映照出多少颗星斗
它见证了多少次残酷的杀戮
它又洗净了多少双罪恶的手
这一切我都无从知晓

我漫步于筼筜湖畔
行走在家乡的桐山溪边
遥看塞纳河左岸的咖啡文化街
聆听莱茵河发源地的涓涓水声
嗅着泥土青草裹着粪便的气味

走到下一座桥就掉头吧，我心想
刺骨的寒风止住了我思古的幽情
也止住了我流水般涌上心间的回忆
转过身
我看到湖上悬挂着大大的圆月

2017.2.12
于筼筜湖畔

自然之光

发表于 2017-02-12   |   分类于 诗歌   |  

享受独自一人的周末时光
房间巨大的落地窗贪婪呼吸着光线
我静静感受明暗每秒每分的变化
窗格木影在书页上悄然起舞

多么宁静美妙的光啊
你们形成与存在于多么久远的纪元之前
我要大声赞美耀眼的日光
也愿温柔歌颂皎洁的月光

我深知，我只是你们万千爱慕者中的一个
过去有许许多多赞美过你们的人
将来还会如此

夜色垂压下来
我沉浸在周身弥漫的黑暗里
等候月光照到我的窗前

了无谐

一盏明灯亮在案头
有如黑暗中指明航向的灯塔

哲人先知们聚会在我的房间
讨论着生命的意义

我看到窗外一种种生活的景象

惯性的人生常使我麻木
麻木中我一刀刀捅向自己年轻激情的梦

下午茶

发表于 2017-02-09   |   分类于 诗歌   |  

没窗子的屋
下午四点该出去走走
看抱小孩的妇女晒温和的阳光
听对弈老翁间迸发不息的马嘶炮鸣

我带着买下午茶的任务
行走间少了几分惬意与释然
成堆食物从货架朝我的购物袋涌来
仿佛迷途的孩子找到回家的路

迷途的孩子还未坐热屋里的椅子
野兽便群拥而入饱腹而散
我捡起茶水间最后一颗烂了半边的草莓

被他厌弃了大半辈子的残缺救了他一命
就像食人族献祭前突然放走缺了食指的国王
有个疯子在坡顶高喊：“草莓，爹爹在这儿！”

2017.2.9
于东区

暗潮

发表于 2017-02-07   |   分类于 诗歌   |  

要等的公交刚好过了一辆
一个女生和我
和空空的站台

索性闲下心来
细细打量起这车来人往的街巷
打量起被晨光涤净了夜的欲望的都市

黎明与黑夜
滚烫的暗潮涌动般
欲望在抬头

噢，公交，你来得再慢一点儿吧
我不愿过早进入只能倚靠白炽灯照亮的室内办公
我想再多看看日光白昼下的一切

二十分钟过去
那个女生和我
和挤满了人的站台

终于还是上车
隧道里灯光交替
公交车门上“黄色区域”四个大字交相投影在车内每个人的身上

哦，该死！
低头打字的瞬间我又错过了什么？
这清晨最真实纯粹的厦门

石板凳上的少女凝神呆望
工地旁的大叔仰头张望
而我的目光正望向他们望着的地方

车内音响突然高声放起第九套广播体操
噢，这前胸挨后背的窄小空间
哪容得下乐动欢腾的青春之舞！

我下车一路小跑
在清醒的清晨
为捕捉易在白日脱逃的诗的灵感

啊，滑稽的清晨
我又悄悄将诗意赶走
大步迈进消耗我过剩精力的方格子小屋

2017.2.7
于上班路上

归途

发表于 2017-02-01   |   分类于 诗歌   |  

西天耀眼的金光燃烧着大气
喷薄的霞光如流水一般
从浓密的阴云堆里倾泻而下
云的后方
浅浅的蓝白色高空那么辽远
仿佛另一个世界

水城威尼斯的波涛
悠悠荡漾在身后
成群的海鸥飞过头顶
那自在的弧线
三三两两
像涂鸦般在宁静古老的天空下任意勾勒

山岗上错落有致的电线杆
丛林卫士般屹立
岩石边微微摆动的芦苇
晚风中低语
一束焰火孤独绽放在大山深处
那儿正有欢庆的人家

车内笑语不断
车外的灯光被速度拉成了线
驿站里沸沸扬扬
古道外轻蹄踏雪
轮子上是移动的家
家和我一路飞驰
故乡在后方
故乡也在前头

一个小小的哲学故事

发表于 2017-01-23   |   分类于 个人随笔   |  

今晚想起了一个哲学家讲物理的小故事，是我以前在书上看到的，颇感受益无穷，不能不说是“听君一席话，胜读高中物理学”呐！就让我以柏拉图的叙述方式来写吧，主要对话人物是苏格拉底和哆啦小默。故事是这样的：

是日，天朗气清，惠风和畅，城镇里勤劳的人们早早起了床，背上锄头铲子，哼着古老又轻松的旋律下地干活去了。哆啦小默也不例外，刚刚填饱肚子，便阔步走向绿油油的田野了。

哆啦小默是镇上地地道道的农民的儿子，为人憨厚诚恳，对有智慧的读书人很是欣赏，也很喜欢和他们聊天，虽然大字不识几个，却一心向往真理，在邻里间也很受乡亲们的欢迎。苏格拉底觉得哆啦小默是有慧根的，在心里琢磨着启发他一下，于是也来到田埂上踱步晒太阳。

正巧，两人撞个正着。苏格拉底说：“哆啦小默，我听闻街坊邻里称道你的为人，并说你心中向往真理，那么我可以认为你是一个渴求知识的人吗？”

哆啦小默闻言，停下脚步，回答道：“可以，我一直都对知识充满渴望。苏格拉底，听你这么问，想必你是有什么东西可以教导我的吧。”

苏格拉底笑着说：“我是个无知的人，我没有什么可以教导你的地方。不过，我突然想到一个问题，你愿意帮我分析一下吗？”

临新春

发表于 2017-01-21   |   分类于 诗歌   |  

停车场的路灯熄了
笼罩这座城市的夜色逐渐淡去
枝条上叶子瑟瑟发抖
厦门也有了冬的冷意

隔着扁平的候机楼
一条条简讯划过
多像头顶闪闪而过的一架架飞机
满载着旅人心底的思念

仰靠座椅
望向天窗外蓝白色的天空
你在白云上方飞翔
我在蓝天下的大地上驰骋

我们都怀揣同样的罗盘
指着回家的路
