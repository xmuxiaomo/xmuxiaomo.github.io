闲话超越

发表于 2017-08-09   |   分类于 个人随笔   |  

近来上映了《战狼2》，上周六我和梦梦一同去电影院看了，感觉不错。

我们可能都有过类似的经验，在一个事件发生前后，许多人往往乐此不疲地在微信之类的社交媒体上制造出一系列配套的内容来吸引我们的注意力，比如娱乐文章、比如小游戏。拿《战狼2》举例，电影开播前后的花絮短片、介绍拍摄过程艰辛的文章、娱乐八卦演员个人生活及演艺生涯的消息、“解救吴京”的华容道小游戏，林林总总，可谓应接不暇，博人眼球。

昨晚GYY在朋友圈晒出67秒“解救吴京”的成绩，我便也点进去尝试一下，结果80多秒才完成。我心有不服，吃罢晚餐，又在家楼下路边试了多次，最后始终突破不了75秒，于是发个哭脸表情回复他，说我只能75秒完成。

望着眼前暗绿色的草场
和金光惨淡的昏黄天空
我的心情是这样沉重与惆怅

你即将离去的消息
令我心乱不已
即使我很平静地通读你的辞呈

无名的沙石堆
几柱冷寂的炊烟
挂在夕阳下的山峰与坡谷
空旷
辽阔无边

相聚总感来日方长
离别始觉仓皇无措
重低音滚滚地在心底咆哮而过

挚友啊
我该为你的决定而欣喜吧
可我为何如此惆怅？

上弦场之恋

发表于 2017-07-31   |   分类于 诗歌   |  

即便是在那一刻
那一个美好时刻的当下
我就意识到失去
正悄无声息地伴随

树影在人文学院苍老的石墙上摇晃
午后三点半的天空晴朗得纯粹
我们刚受完音乐观念的洗礼
背着书包并肩从楼里出来

清爽洁净的冷风吹面不寒
秋季的上弦场依旧绿树成荫
我们火焰般炽热的爱情夏日
在明媚的秋风中轻柔踱步

这是青春年华里最美的时刻吧
我心下不由感慨
我旋即调动全身所有的感官沉浸于此
感受这美得令人心碎的
无可奈何的
失去

黄昏渡

发表于 2017-07-28   |   分类于 诗歌   |  

日头还未西斜入海
我与夏风一起坐在岸边远眺
夏风好不安静
不时逗弄着暗青色的水波
和我双膝前粉色的三角梅花瓣

渔人叼着一支烟
悠闲地倚在不远处垂钓
海鸥低低地飞梭于铁桥与水鼓之间
穿蓝布衣衫的船工忙碌地粉刷桅杆
阳光悄悄从身后把他变成了金人

水波呵，鼓浪屿
我从不曾这样久久地将你眺望
在夕阳肆虐的黄昏里
在浅浅的海峡边

2017.7.28
于和平码头

漫步随感

发表于 2017-07-26   |   分类于 诗歌   |  

夜风无比柔和
没有了白日灼人的暑气
我们放慢脚步
月色轻盈的校园如此可爱

曾经
芙蓉隧道的两头
系着甜蜜而青涩的思念
那时，我还不会写诗
只顾埋头穿行其间
只是固执地守候在
霞光满天的木棉树下
隔着生冷铁栅
遥望石阶的顶端
那个
你将会走来的
梦的方向

今夜
我像往常一样
牵着你温热纤柔的小手
漫步在青石道上
月色融化我们的蜜语
化成晚风中缱绻不去的眷恋

我们的告别
依旧隔断在铁栅的两侧
唯有唇边还清晰留存着
一个吻的余温

2017.7.26
于街灯下

无心准备
像去年的七夕

空调安静地催眠
饱腹之人皆已昏昏睡去
午后的办公室
空空荡荡
只剩几米撒野的阳光
跳跃在软软的浅灰色地毯上
不肯离去

我依旧俯首
读孤单的诗集
背对大山和草场
圈圈写写
任思绪与意象穿梭我的身体
像风一样
轻柔无端地蔓延

忠贞不渝的奴仆
搅扰多少清梦和欢快的真实
踉踉跄跄
小丑般唤醒鼠标与键盘

起身寻找太阳清澈的影子

窸窸窣窣
喀嗞喀嗞
叽叽喳喳
瞧！
动物农场复苏了

两种风景

发表于 2017-07-11   |   分类于 诗歌   |  

清淡的白云
游移在倾斜的地平线上
大片绿色的草场
拼接地毯似的无限延展

你看飞驰铁道旁的风景
怎料那农家的炊烟正巧飘进我的鼻里

暖烘烘的岩石
幸福也许在正午时分栖身于此
苍茫的天际
光线变幻得清澈而随意

我像不尽山巅的一棵野树
平静等候落日沉入繁星的璀璨

大山深处没有明丽的色彩
却有几分黯哑的孤独
我们沉醉两种风景
在同一个时刻

2017.7.11
于北辰山
