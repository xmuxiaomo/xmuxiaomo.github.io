祈光

发表于 2017-03-26   |   分类于 诗歌   |  

直到听见清晨的第一声鸟鸣
我才真正从梦中清醒
窗外的天空蓝得深邃
依旧浓密的夜色笼罩千家万户
这城市多么宁静

我在夜里进入到第四重梦境
梦境里我的门牙掉了三颗
退回梦的第二重
我斗胆令舌尖向前
触碰到的坚实令我既惊讶又心安

白日在博物馆里看尽了千年
夜里难免于梦中止不住思索

我呆坐在窗前
多希望光明快一点来临

清净寺

发表于 2017-03-24   |   分类于 诗歌   |  

蔚蓝天空下
古镇小道旁
城墙将影子投在斑驳的石砖上
北宋的院落格外清净

高高的石柱表面满是风过的印迹
阳光一如既往地卧在青草地里熟睡
我闭上双眼久久地伫立
仿佛石柱般已沉默了千年

曾经的辉煌像时间一样老去
我无从知晓这儿究竟发生过什么
平静的空气中
唯有久违的桂花芬芳馥郁

木棉树抖落层叠的记忆
刺桐城天色将晚

读《阿古利可拉传》小感

发表于 2017-03-23   |   分类于 读书笔记   |  

用两个晚上的时间阅读了古罗马塔西佗的作品《阿古利可拉传》，心中感悟良多。我记不住书中提到的许多帝王将相的名字，但对阿古利可拉的中庸之道，对其行事的谨慎与谦和是无法忘怀了。

书中有这么一句话：“在那个时代，美誉和恶名有着同样的危险。”的确如此啊，自古以来人类的天性里就有嫉贤妒能的特点，恶名者被世人唾骂是很常见的，但是美誉为何危险？就是因为享有美誉之人周围不乏小人，小人心胸狭小，见不得别人的好，阿古利可拉在任期间，为罗马攻占不列颠立下不朽的功勋，但是当时的皇帝多米先就处处堤防与担忧，害怕大臣的光芒与荣耀高过自己。不过阿古利可拉实在是谦和君子，他的穿着与外貌及其普通，引不起身边之人的注意，丝毫没有立大功者的自豪与骄傲，这也是他为人处世的大智慧之所在啊。多米先是想挑刺也挑不出来，只好派人多加注意阿古利可拉的行动而不敢再有所动作了。

不列颠方面，一位名叫卡尔加库士的酋帅在集结兵马、号召天下勇士以抵抗罗马的进攻时，作了一番振奋人心的演说。演说以自由为最高宗旨，不愿为奴的精神在听众间引起巨大的反响。这篇演说非常棒，我想把它摘录在此处：

周日小记

发表于 2017-03-19   |   分类于 个人随笔   |  

又到美好的周日时光，我和梦梦依旧出发去“翔安度假村”。最近这几个周末，我们时常会在游客挤满思明校区的时候，悄悄出逃，自驾前往翔安校区，那儿地广人稀，白天在校园里几乎看不到人和车，很有到了另一个国家甚至另一颗星球之感。大大的校园里，仿佛只有我们两个人，阳光明媚也好，小雨点点也好，都十分惬意舒畅。

周末，我们喜欢换个环境生活一天，距离的拉远，让我们远离日常的琐碎事务，让周末时光更加纯粹与静谧。在食堂三楼吃碗不辣的麻辣烫或一丁点冷辣的凉皮，在校内的万达影城看个新上映的电影，在图书馆的自习室安静地看书写作，无一不让我们感到开心与愉悦。

今天中午吃过午餐，我和梦梦说：“你将面临一个严峻的考验”，梦梦眨巴着大眼睛问我是什么考验，噔噔蹬蹬——练车！坐上车，调好座椅和后视镜，我便爽爽地坐上副驾驶座，当起教练来。其实梦梦拿驾照的时间比我还早一年，只是后来开得很少。不过车技学完一般也不会遗忘，操练操练，但手熟尔。梦梦开得很认真，我坐得也很放心，她说想练习一下沿环岛绕圈，我说好啊，结果梦梦练得起劲，一圈接一圈，直到我头晕目眩，她才笑嘻嘻地从一个出口转出去。

手机

发表于 2017-03-19   |   分类于 诗歌   |  

手机此刻正握在我的手里
我要用它来赞美它自己
五十年前的人们想不到它今日的普及
那时它甚至只是渺远未知的概念

空气是无形的介质
我们看不到被叫作信号的东西
邮递马车在斜阳下驮着书信蹒跚远行
书信却在指尖轻盈迅捷地流转

亲人挚友那远自故乡的问候
即刻便直抵我们心间
时空因手机而扭曲

我放下这神奇的小物件
推开窗子
悲伤与喜悦布满了整个天空

轻飘飘的生活

发表于 2017-03-16   |   分类于 个人随笔   |  

回想大半年来的生活，轻飘飘的，许多时候像不曾存在过。这半年多来，做过的和经历过的事儿不少，但是大多没有什么实在的意义，在我的印象里也十分模糊。

我细细想来，真的能让我觉得自己生活过的事儿，只有珍惜和爱护我的女朋友，真心地与家人相处，写过的诗和文，行过的路。此外，再无其他。

看着堆满卫生间的衣裤和袜子，看着厨房水池里泡了不知几个星期未洗的碗筷，看着架子上摆了一两年仍未拆封的书籍，我突然看不见我生活过的痕迹，我的时间都去哪儿了？我曾经的想望都去哪儿了？我恪守的原则与底线都被抹擦净了吗？我曾经希望的生活与如今又相差多远？

这个社会很实际，人与人间的关系，说白了，除去亲朋与挚爱，真的只有利益关系。说到麻木的“谢谢”与“辛苦了”并不能代表一个旁人是在真的关心你，他不过是鼓励你继续帮他的忙而已，无休无止。如今的工作除了压榨时间与消耗过剩的青春热量外，我感受不到还有什么别的。一切都太轻了，太虚无了，遑论什么人生的价值与理想。我的印象里渐渐没有了白天，我有的只是灯光，嘈杂的人声，食堂的饭菜，川流不息的街道，无尽流转的黑夜与霓虹。

我得到的是可以糊口的钱财，微不足道的让我觉得堕落的所谓“自我提升”，我失去的是宝贵的时间，是生命的沉思，是梦想与憧憬，是健康的身体。

沉默在呐喊

发表于 2017-03-15   |   分类于 诗歌   |  

海鸥有翼
可以在空气中翱翔
我有脚
可以在地面上行走

哦，海鸥也有脚
她可以轻点水面
也可以沿着树根走
而我张开双臂却无法飞翔

我们都是天地间一团形迹可疑的物体
海鸥不知道我所谓的生活是什么
她不知道我究竟是个什么东西
就像我不能真的知道她一样

每个物种无法切身了解另一个物种
甚至像我，一个男人
无法去了解女人
除了我之外，我不了解任何东西

天空以雷霆万钧之势
揪住我的衣领厉声质问我
你真的了解你自己吗？
我愕然失语

我问充满智慧的流云
我究竟是什么？
可流云啊，你却默不作声
为何你不肯向乞讨意义的人施舍一束光柱，或一场大雨？

浪花一朵接着一朵
那是大海无穷无尽的追问
岸边的岩石是位古老的智者
他回答以永恒的沉默

2017.3.15
于轮渡海岸

我想我会爱上这个地方

发表于 2017-03-13   |   分类于 诗歌   |  

我想我会爱上这个地方
这儿远山环绕
这儿的绿树合抱着每一座青峰
这儿的道路无限延伸
让我忍不住想尽情奔跑

这儿的牛儿三三两两
波光粼粼的湖面上白鸭成行
太阳赤着脚
和村民一起躺在树荫下小憩
清新的风将树梢上的缕缕诗意带到每一扇窗前

这儿的房子不高
这儿到处洋溢着土地的欢乐
这儿的恬静是一首风琴诗
这儿的野趣是一幅饱含十四种鲜明色彩的油画
这儿的溪流正弹奏着一段唯美写意的乐章

噢，这浑然天成的美丽
这大山深处的气韵
我究竟该用怎样的词语来形容你？

我想我会爱上这个地方
哪怕我只是刚刚踏足

2017.3.13
于北辰山
