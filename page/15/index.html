论成就感

发表于 2017-05-15   |   分类于 个人随笔   |  

大约四个月前的一天，我在上班时，突然想到了一个问题——我们所追求的，归根结底，是不是成就感呢？

由于当时手头有事，没能细细思考这个问题，我只简单地把这句话发到说说上，用于记录这闪过的念头。我以为这个问题很可以仔细玩味一番，所以当晚女友想和我聊这个问题时，被我婉言拒绝了，因为我想不带偏见和一些先入为主的观念来独自省察和体悟这个问题。

心理学上有“未完成事件”的说法，意思是想到或想做的一件事，如果拖了很长时间一直没有完成，此事便会长期盘踞在心间，占有大脑一部分思维的空间，在不经意间总会想到这件事，这就是未完成事件。直到这件事被重新拾起，认真做完，它才会慢慢淡出你的心间，变为不那么重要的东西。我想，这个问题在四个月的时间里，一直是我心中的未完成事件。时间好快，又或者是生活实在太过丰富，我的注意力十分稀缺，时而被这个吸引，时而被那个吸引，满眼的新鲜事物诱惑着我，让我没能静静思考这个问题。

旁的话就不多说了，在思考成就感之前，我想先谈谈感觉是什么。我们是绝对离不开感觉而生存的，这一点毋庸置疑，我们难道不是时时刻刻都在感知和触及这个世界吗？眼能看到各种色彩与形状，耳能听周围存在的声音，口能尝物体的酸甜苦辣等滋味，鼻能闻香的或臭的气味，皮肤能接触或坚硬或柔软的不同质地的东西。我们张开身体的每个毛孔与每寸肌肤，运用感官来感受我们所处世界的一切能被我们感知到的事物。当然，这是我们最基本的感觉，往更高一层次来说，我们能感觉到同伴的心理活动和他人没表达出来的一些言外之意。虽然在感受这些他人内在活动和情感的过程中，我们难免会基于自己多年生活的经验和主观猜测来进行感知，也就是心理学上所讲的“投射”。因为归根结底，我们是无法准确知道他人在想什么的，许多误会往往会在这个过程里产生。我们有了这许多的感觉，也才更了解何为自身的存在。因为我习惯了有感觉的生活，所以我无法想象和体会到没有感觉的生活是什么样子的。没有感觉的时候我还会不会思考感觉这个东西？其实感觉远远超越于语言之上，因为即使我不懂得有“感觉”这个词语，我不懂“感觉”这个词语有何对应的意思，但我也还是会有感觉的，那是感觉的元，它不需要语言来描述与解释。就像一个不会任何语言的动物，但是它还是有感觉在，它会感到饥饿，它会感到冷，感到痛。语言只不过是我们约定的一种同类之间交互信息与彼此沟通的媒介罢了。事物的存在本身是超越于语言之上的，语言的语汇再多，也道不尽我们认知所能到达的领域与高度，此可谓“言有尽而意无穷”，意和认知总是领先于语言的，言语有它的尽头，而意思则可以无限延伸。

把所做的每件事都当成作品来完成

发表于 2017-05-06   |   分类于 个人随笔   |  

我喜欢把正在做的每件事都当成作品来完成，正如现在写下的这些文字，字字句句都由我丰沛的心灵所流淌出来。我很喜欢在空闲的时间里写点儿什么，虽然有时并没有想好要写什么，但是安静会催生出许多意想不到的东西，静静流淌，无喜无悲。

我刚泡了杯咖啡，像无数个午后一样，虽然重复的次数很多，但每次都是新的感觉与经验。烧开水，洗杯子勺子，撕开一袋新的咖啡，将咖啡粉小心抖落在杯中，冲入滚烫的开水，搅动勺子，闻着动人心魄的香气，摆在桌上，陪伴我工作或是学习。我早已不简单地以为这是一杯咖啡了，它是每个午后的期待，是我内心中对每个下午美好时光的一份期待。咖啡不仅使我精神更佳，重要的是它常常为我带来创作的灵感与诗意。我搞不清闻到咖啡的香味时，我的生理和神经经过怎样的活动，我大脑的哪块区域变得比平时兴奋和富于创造力，只是我常常在一杯咖啡的时间里，经历许多美好的想象中的畅游，发过很多美好又单纯的呆，写下许多过了就难再提笔去写的文字。我想，这才是咖啡对于我寻常日子的意义，我亦把每次泡咖啡都当成一件艺术品来完成，乐在其中的感受，实不足与外人道也。

回家真好

发表于 2017-05-01   |   分类于 个人随笔   |  

明朗的阳光照在浅蓝色的窗帘上，鸟儿在枝头叽叽喳喳欢叫，我心下一暖，又是美好的一天。

在家的日子真是安静惬意，门前的小树郁郁葱葱，河水平静地一天天流淌。远山低矮，却年年环绕着小城，依偎相伴。

这次五一回家，我和姐姐、姐夫搭乘同一趟动车，不过我们不在同一个车厢，因为买票时间不同，座位的安排也就被打乱了。但列车也就是八九节车厢，所以即便不在同一车厢，我们的距离也不很远。列车往家的方向疾驰，我困了就靠在椅子上呼呼睡去，醒来就看一会儿野夫的《身边的江湖》。野夫的这本书我大多只在路上看，总觉得很符合旅途的心情。我们将近九点钟出发，三个多小时后就到达福鼎站了，爸爸每次总会在出站口等待，等待着儿女平安回来。妈妈平时也常来接我们，偶尔没来便一定是忙着在家里准备一桌美味可口的饭菜。

爸爸知道我喜欢开车，见我先出站，便把钥匙交给我，让我把车倒好位置，他则继续等待姐姐和姐夫出来。每次开爸爸的X3，我都倍儿加留心，因为爸爸平时非常爱车，隔三差五地总要把车擦洗得纤尘不染。爸爸基本上都是自己洗车，一方面是觉得自己洗会更注意细节，另一方面也当是检查检查车身有没有细微的受损。爸爸很爱护车，我在驾驶爸爸的车时便也格外认真。不一会儿我们就到家了，爸爸张罗着我们在客厅喝茶，妈妈说饭菜马上就好了。呼！回到家的感觉真好。

朱颜存

发表于 2017-04-26   |   分类于 诗歌   |  

柳絮飘飞的四月
我又来到故宫
时隔二十年
往昔的足迹早已无处寻觅

黑枝绿叶蓝天
白水红墙黄瓦
青铜狮子依旧镇守在太和门外
十八樽象征权力的大鼎仍纹丝不动地立在原地
日晷与嘉量
昼夜不息地丈量着时光
那远去的皇权慷慨打开了宫殿的大门
寻常百姓得以一睹天子的居所

与人有关的一切都渐渐融解在金黄的岁月里
诏书御旨黯哑了声响
编钟安静地陈列在展柜
文武大臣不见了踪影
保和殿里的状元如今已无人过问
储秀宫中妃子美人们的笑语恐怕只有院中苍老的古树记得吧
铜铁大缸已几十载不再蓄水
上了锁的木门更封住了往事几许

钟表馆的机械纹理间尽显前人智慧的高度
一跪一笔都在诉说百年前动人的小事
海屋添筹和群仙祝寿
乾隆为他母亲准备了六十份生日礼物
铜壶还滴水吗？
那滴定的标尺已远离了浮沉乱世

呼啸的大风洞穿了重重木门
历史蹒跚行走在汉白玉石桥

2017.4.26
于御花园

学旅

发表于 2017-04-25   |   分类于 诗歌   |  

用三天时间
为生活推开一扇新窗
没有雾霾的侵扰
京城天朗气清

滚滚而过的幻灯片
散发知识光芒
那凝聚的心血化为森林里春雨点点
鸟儿得悠然俯瞰新绿

一块块要点像一个个圣地
导游将行程排得紧凑
屈指三日即览遍蜃景
那鸟儿落入林间却已不再迷路

证书不过是山顶一块极其普通的石子啊
年轻人皱着眉从苏格拉底手中接过
这难以掩盖的失望
已然将美妙的登顶历程抛掷得一干二净

时间晃然进入倒数
我只身穿透沙漏的孔隙
我像初生的婴儿般睁大双眼看周遭的一切
掌心却握不住一粒沙子的流逝

时日恒常

发表于 2017-04-24   |   分类于 诗歌   |  

没有多余的话语
空寂的图书馆里只有风扇换气的声音
磁拉磁拉
高高地悬浮在头顶

当几百张嘴巴同时合上
便有这几百个的灵魂在此处游荡
灵魂的游走静静悄悄
书页翻动的细微声响好不热闹

照明灯打亮面前的书桌
仿佛灵魂将全部的注意力倾注在了纸上

厚厚的典籍是年老的智者
暗暗默默
这无数的智者怀揣动人的故事
等待后人们愿意开放的心灵

我们是前人的后人啊
先前还将前人的言行看进眼里
转眼便发现自己已是当初的前人
而后人们
正像我们曾经看待前人那样
直勾勾地看着我们

已故之人何所寻？
去眺望苍茫的星空
去守候在黄昏低矮的墓碑前
去翻看旧日的相片
去到记忆里找寻

怎么，怕这些都不够真实吗？
怕找到的只是记忆中模糊支离的细碎光影吗？

那打量打量他们的灵魂与精神吧
在沉沉的书页间
在浅浅的字迹中
这儿有他们响亮的声音
有他们明丽的青春年华
有青春年华里丝丝缕缕的惆怅
有宁静甜美的爱的心跳
有失意落魄时愤怒的呐喊
还有，
还有我们平凡单纯的每一个日子

生命的痕迹不会永远消失
生动的日子都还停留在那个时刻哟！

高高地悬浮在头顶
那些似乎逝去了的一切
在宇宙深处
磁拉作响

2017.4.24
于国家图书馆
