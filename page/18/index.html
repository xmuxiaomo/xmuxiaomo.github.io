沉默在呐喊

发表于 2017-03-15   |   分类于 诗歌   |  

海鸥有翼
可以在空气中翱翔
我有脚
可以在地面上行走

哦，海鸥也有脚
她可以轻点水面
也可以沿着树根走
而我张开双臂却无法飞翔

我们都是天地间一团形迹可疑的物体
海鸥不知道我所谓的生活是什么
她不知道我究竟是个什么东西
就像我不能真的知道她一样

每个物种无法切身了解另一个物种
甚至像我，一个男人
无法去了解女人
除了我之外，我不了解任何东西

天空以雷霆万钧之势
揪住我的衣领厉声质问我
你真的了解你自己吗？
我愕然失语

我问充满智慧的流云
我究竟是什么？
可流云啊，你却默不作声
为何你不肯向乞讨意义的人施舍一束光柱，或一场大雨？

浪花一朵接着一朵
那是大海无穷无尽的追问
岸边的岩石是位古老的智者
他回答以永恒的沉默

2017.3.15
于轮渡海岸

我想我会爱上这个地方

发表于 2017-03-13   |   分类于 诗歌   |  

我想我会爱上这个地方
这儿远山环绕
这儿的绿树合抱着每一座青峰
这儿的道路无限延伸
让我忍不住想尽情奔跑

这儿的牛儿三三两两
波光粼粼的湖面上白鸭成行
太阳赤着脚
和村民一起躺在树荫下小憩
清新的风将树梢上的缕缕诗意带到每一扇窗前

这儿的房子不高
这儿到处洋溢着土地的欢乐
这儿的恬静是一首风琴诗
这儿的野趣是一幅饱含十四种鲜明色彩的油画
这儿的溪流正弹奏着一段唯美写意的乐章

噢，这浑然天成的美丽
这大山深处的气韵
我究竟该用怎样的词语来形容你？

我想我会爱上这个地方
哪怕我只是刚刚踏足

2017.3.13
于北辰山

辗转

发表于 2017-03-10   |   分类于 诗歌   |  

思想的笔在脑中咆哮
白纸的荒漠需要词语的墨水

时间重重压住我的双肩
脖子渴望舒适的倚靠

我用清水搓脸
才恍然发觉脸皮是如此之薄

虚无在白昼里蔓延
脊柱忠实镇守身体的花园

雨解开了天空的扣子
为给大地织一席锦衾

我吐出的一小团叹息
旋即凝结在床头

夜在枕上沉沉睡去
而我仍在追赶梦的列车

天空

发表于 2017-03-04   |   分类于 诗歌   |  

在暖意尚未抵达的三月初
我喜欢看傍晚六点钟的天空
太阳收敛起耀眼的光芒
天空只留下淡淡的黄蓝色光亮
我还看不到群星
因为天空还没换上夜的披风

天空很忙
在轻飘飘的云的上方
是一道道飞行的航线
天空很高
有飞鸟也无法越过的空间
天空深浅不一
有比浅更浅的远方
也有比深更深的遐想

我听不见天空的语言
也不知道此时有谁和我一样在望着天空
会是游吟的诗人吗？
会是沉默的垂钓者吗？
会是醉眼看世界的乞讨者吗？
会是远走天涯的浪子吗？
会是仰头问天的青年哲学家吗？
又或者，是一个穷困潦倒的画家吧

风将云层吹得紧密起来
大地已华灯尽染
我喜欢看傍晚六点钟的天空
我享受天空赠予我的无尽幻想
我大概能够理解天空
因为我拥有和天空一样广阔的胸怀

2017.3.4
于十一楼阳台

甩锅文化

发表于 2017-03-04   |   分类于 个人随笔   |  

夜里关了手机，一宿无梦，睡到自然醒，醒来一切都那么安静，好久没有这般舒爽感受了。近来大多数时间都是闹铃将我叫起，噢，又该上班了！过去的这二十天里，由于公司项目赶进度，上了十七天班，只休息了两天，今天是第三个休息日，而且其中大概有十天晚上都加班过了九点，实在是疲惫不堪。回家倒头就睡，上班久坐不起，日子久了，头昏脑涨，并感觉腰部有点疼痛。

确切来说，不是有点儿疼痛，而是年后到现在几乎每晚睡觉都会痛醒。是福是祸，这说不准，爸妈比较担心，说让我有空去医院检查检查。我倒觉得这是个“好毛病”！大概每晚睡到六个小时左右，就会开始疼痛，这样既有了之前平稳的睡眠，又可以早起，而且这毛病也怪，起来一会儿人就好好的了，没有任何不适感，就睡觉时才有，也许是命运之神在提醒我，该早起啦！

哈哈，言归正传，扯了一堆闲话，该说说近日学习到的一种新公司文化了，我将它取名为“甩锅文化”。顾名思义，就是将背在自己身上的锅丢给别人，锅且不论黑白，只要一接到手上，迅速丢出去，自己一身轻松。起初培训时听了不少介绍公司文化的讲座，独独对此“甩锅文化”没有耳闻。为何我近日发觉到了呢？如你所想，我定是有许多亲身体验的。

巷弄

沿不太陡的小径往上走
渐渐远离闹市
脱去嘈杂的外衣
脚下便有轻风

旧宅沐浴和暖的阳光
电线胡乱地搭架
蔬菜刚刚入锅
窄窄的小巷幽幽为我带路

曲径狭长望不到尽头
不知其通向何处
唯闻钢琴声星星点点
仿佛溪水静静地流淌

每每迈步向前
儿时老家的记忆就愈发清晰
老家近旁也有窄窄的小巷
黄昏是一支金色的歌曲

画笔用心调配大自然的色彩
远山深浅黑白影影绰绰
羽毛球飞上落满霞光的屋顶
再不愿醒来

日记本

发表于 2017-02-26   |   分类于 诗歌   |  

夜晚读书写作时
书桌上总有你最甜美暖心的微笑
陪伴我陷入深深的迷惘
也陪伴我写下对生命的每一次感悟

翻看日记本
无意间发现
原来你是我诗意的启发者
你将优美的诗与我分享
那潜藏的诗歌的种子
不经意中已在我的心里长出小小的嫩芽

读着曾经写下的我们之间的点点滴滴
回忆浸湿了我的眼眶
那是美的令我思念的感动
往事那么真
一幕幕在字里行间漾开
一笔笔流淌出心灵的无限喜悦与沉醉

你说感谢我记录下这许多美好珍贵的瞬间
可你却不知道，正是你啊
我亲爱的恋人
是你，让我把生活过成了诗

我奢想，我有我们全部的记忆吧？
哦不！我只拥有一半的记忆
还有一半该在你那儿
又或是
我们都拥有全部的记忆
关于我们爱情全部的记忆

这日记是秘密的
这是只属于我们两个人之间的秘密
可我担心，它终有一天会成为公开的秘密
有另外一颗心灵在另外一个时空里
开启这岁月的封条
那时，一定有轻柔美妙的乐曲从匣子里潺潺流出

2017.2.26
于家中
