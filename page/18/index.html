永恒

发表于 2017-02-22   |   分类于 诗歌   |  

盆里的假花永不凋零
灰尘却蒙住它心灵的翅膀

我问铺满石头的路
多少惶惑与焦急曾在这里走过

海风中舞步轻盈的百岁老者
她面庞纯净一如往昔

心跳声和着秒针滴答此起彼伏
生命的一瞬止于鼻尖

我团身坐在小小的屋檐下
思想的光追逐抵达深邃冰冷的星际

词语忘却其含义
满地的诗稿落叶般凋零

我摘下时间的面具
噗通，月亮掉进井里了！

2017.2.22
于家中

门外的夏天

发表于 2017-02-18   |   分类于 诗歌   |  

我家门前有条河
不宽不窄
一年四季静静地流淌
日子也一年一年徐徐流淌着

天空明朗
午后的阳光暖烘烘地照耀大地
地面反射的亮光刺得我眯缝起双眼
一副慵懒模样

出门沿河漫步
道路宽阔河水悠悠
脊背略微流汗
夹杂湿气的热风股股从河面上方吹来

弧形右转弯的转角处
有个穿纯白短袖上衣红裙子的女孩
在黄绿色的大大遮阳棚下卖熟透的西瓜
霎时一阵清凉扑面而来

啪嗒啪嗒
拖鞋带着我缓缓挪步摊前
她眼带笑意好像春风，我知道
她的名字叫夏天

众生相

发表于 2017-02-16   |   分类于 诗歌   |  

周围有窗
却全被人为关上
窗帘胶带般缠绕
办公室被封为密不透风的方盒子

方盒子里是包好的精致的饺子
一百来个，一个紧挨着一个
微波炉一样的嗡嗡声萦绕耳畔
持续升温的空气蒸得我头昏脑涨

外面的阳光照不进来
里面的人惬意地晒着灯
一个饺子状的人在墙角打了个喷嚏
该死！满盒子的饺子都馊了

整齐的镜片一动不动
盯着持续亮着的计算机屏幕直至年华老去
墙上“只争朝夕”四个大字熠熠生辉
值得我珍惜的却在盒子外边

围城不在书中
阳光灿烂的日子
城里的人
从来不曾存在

2017.2.16
于总行

五岁

发表于 2017-02-15   |   分类于 诗歌   |  

无需任何言语
一张老照片就让我感动

吹熄生日蛋糕上的五支蜡烛
二十年一晃而过

时间酿出芳香的回味
我品着旧日圆桌上还未喝完的半碗热汤

那个天真的孩童真的是我吗
他身边的亲人如今去了何方

照片让记忆凸显
珍贵金子般收藏于心底

橱柜和灶台毫不陌生
因我无数次在梦中将你们记起

好想抱抱曾经的自己
孩子，你已长大

时间的错觉

发表于 2017-02-14   |   分类于 诗歌   |  

我们相识之初
时间流淌得很随意
一群同学围坐在偌大的教室里听讲
我们偶尔交流
却并不刻意
铃声响起
转身那样轻易

后来
时间变得很悠长
你托着腮凝望黑板
我的目光却不自觉地瞥向有你的窗前
窗外树影斑驳
我呆呆地盯着你手中的笔出神
又常悄悄移动视线
望向你的侧脸

牵你的手
在月色如火般静静翻涌的夜晚
时间仓促地退到地平线以外
我乱了方向
足迹一遍遍将芙蓉湖圈起
寒风吹过脸庞
你手心的温度那样炽热

再后来
时间变得很慢很慢
慢到一日便盛开出一朵花
慢到细沫从浪尖高高腾起
慢到每一秒都像永恒
慢到每一刻都芬芳馥郁
我将每一天
如花瓣般藏进书页

有你的日子
时间变魔术似的忽快忽慢
快时一吻天便老去
慢时我就陪在你身旁
从早晨到黄昏
从郊野到书房
澄净的天空中有米色的阳光

如今
时间从层叠的叶缝间滑过
打碎一地光影
老旧街区的十字路口
我举目四望
越过时光的轨迹
我看到
满满都是暖心的故事

2017.2.14
于老街树荫下

夜逢千杯少

发表于 2017-02-13   |   分类于 个人随笔   |  

零点二十分，睡意全无，我知道即使此刻关了灯躺在床上，自己也久久不会入睡，因为大脑很活跃，浑身的细胞也都苏醒着。

本想蒙头睡去，明早四时起来再写这篇文字，但是兴之所至，情绪到了，无法不写点儿什么。怕明早起时，昏昏沉沉，是啊，过了这阵子，也许再难提笔写下此刻的生命历程与感受。我很珍惜和享受这种意欲书写的快感，愿自己常能有此“踏破铁鞋无觅处，得来全不费工夫”的随性之感。这种感觉是妙手偶得的，是可遇而不可求的，是来临时欣喜而毫不抗拒的，是充斥全身而又十分微妙的。

傍晚下班，在“子母粉丝”店匆匆吃了碗牛肉粉丝，手机屏幕上弹出一条消息——“厦航MFxxxx航班将提前二十分钟抵达高崎机场”，赶忙扒两口粉丝，喝两勺汤，起身便出发去机场。今晚梦梦回来啦，随机的还有她的两位老师。送他们都回到了住处，我便安心赴友人Y之约，到家楼下的“每天咖啡”门口等候。

Y很守时，两分钟后便出现在眼前，我们有一年半多没见面，但却一眼就认出了彼此。

流水

发表于 2017-02-12   |   分类于 诗歌   |  

不尽的湖水在晚风中
汩汩向前流淌
这水行走过的世界一定比我更辽远
我猜不出它流经了多少地方
流走了多少个世纪

身旁的流水流过多少人的身旁
它映照出多少颗星斗
它见证了多少次残酷的杀戮
它又洗净了多少双罪恶的手
这一切我都无从知晓

我漫步于筼筜湖畔
行走在家乡的桐山溪边
遥看塞纳河左岸的咖啡文化街
聆听莱茵河发源地的涓涓水声
嗅着泥土青草裹着粪便的气味

走到下一座桥就掉头吧，我心想
刺骨的寒风止住了我思古的幽情
也止住了我流水般涌上心间的回忆
转过身
我看到湖上悬挂着大大的圆月

2017.2.12
于筼筜湖畔

自然之光

发表于 2017-02-12   |   分类于 诗歌   |  

享受独自一人的周末时光
房间巨大的落地窗贪婪呼吸着光线
我静静感受明暗每秒每分的变化
窗格木影在书页上悄然起舞

多么宁静美妙的光啊
你们形成与存在于多么久远的纪元之前
我要大声赞美耀眼的日光
也愿温柔歌颂皎洁的月光

我深知，我只是你们万千爱慕者中的一个
过去有许许多多赞美过你们的人
将来还会如此

夜色垂压下来
我沉浸在周身弥漫的黑暗里
等候月光照到我的窗前

了无谐

发表于 2017-02-11   |   分类于 诗歌   |  

一盏明灯亮在案头
有如黑暗中指明航向的灯塔

哲人先知们聚会在我的房间
讨论着生命的意义

每本书里都藏着一扇窗
我看到窗外一种种生活的景象

惯性的人生常使我麻木
麻木中我一刀刀捅向自己年轻激情的梦

胡诌八扯的人生观太多
说久了好似他们都懂该怎样实现生命的价值

我是聋子也是哑巴但不是瞎子
让那些肤浅无趣的观念统统都见鬼去

我依然有改变世界的梦想
隐于这首凑够了十四行的诗里
