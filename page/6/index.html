生活在习惯里

发表于 2021-03-09   |   分类于 诗歌   |  

离开，一次又一次
告别一个地方
宿舍，老家，后门的斜坡
告别习以为常和以为的长久
物件的摆放
毛巾，牙杯，厕纸的角度
视线所及的一切如常

当下的麻木、茫然
甚至还抱怨生活乏善可陈
而当离开，如一位老者向我走来
将一切实在吹入梦中
梦境的虚幻啊
我在黑暗中徒然地伸手
翻寻记忆，翻寻阵阵空无
直到下一个实在
替代了从前

后知后觉如我
才意识到每次离开
真实得有如幻境
为何依恋总也如此后知后觉？

而我仍旧是我
一切境遇的执著构筑者
物件的摆放
规律的时点
清晨窗外喷薄的日出
和我口中哼起的歌

啊，原来我一直生活在自身深处
生活在小小的习惯里！

2021.3.9
于家中

思索时间的紧迫感

发表于 2021-03-08   |   分类于 个人随笔   |  

工作久了，休假时的节律也受到影响，周一到周五工作日时间，便会感到一定的紧迫感，觉得不能彻底自在放松下来，似乎与工作、单位、同事还有一种联结，有种随时可能被打扰的惊慌。而周末，虽与陪产假重叠，但就很放松，早晨也睡到比较晚才起床，似乎天下共休，才颇有假日的氛围。啊，工作的毒瘤！

为何每个假期临近尾声时，都有种时间的紧迫感，不论这个假期是长是短？为何每个阶段结束时也有这样的感觉，比如本科毕业，研究生毕业？是否我们沉溺童年无忧无虑的时光、学生时代单纯快乐的时光，而不愿走入繁复庸碌的社会，不愿被世俗的力量沾染，不愿自由受到限制、时间无法支配？是否我们惧怕这种存在的无力感？是否我们直到临近死亡，都难以摆脱这种时间的紧迫感，难以在所处的当下长久地保持惬意、散漫、随性，诗意地栖居？而在事物的开头和起初阶段，没有这样的感觉，是否因为充足、未知、无知，抱着尝试的心情，尝试新的事物，尝试另一种生活，尝试存在的这仅仅亿万分之一的可能？

也许咖啡正合适

发表于 2021-03-06   |   分类于 个人随笔   |  

念头和欲望真是搅扰人心……这几天休陪产假，在月子中心读了加缪的《局外人》，感觉自己也想尝试写小说，念头一起，便连夜构思，用笔圈圈点点，在书本末几页写下了诸多拟加入小说的成分。

构思的内容与近日经历有关——“皮蛋”出生了！所以想写一篇题为《新生》的小说。小说当然不是直接写我的个人经历，而是要有主人公，有小说的形式，有描写有思索，有关注的社会或心理现象。思前想后，虽然新生儿、初为人父是天底下大多数人都会有的经历，写下读来自会引起许多共鸣通感，但这仍不足以成为一篇好的小说的题材。怎么说呢……考虑良多，觉得我不爱写多余的文字和废话，我珍惜一字一句的分量，正如生活中的我寡言少语、惜字如金……

所以诗歌适合我啊，就像咖啡，温热得刚好，不长的一杯的时间就足矣。小说则像丰盛的正餐，是文学的另一个世界。我爱读小说，体验到沉溺其中的阅读的乐趣，这不就该感到惬意与满足了吗？

借我一个安心入睡的下午时光

发表于 2021-01-02   |   分类于 个人随笔   |  

元旦假期，连续两个下午，四五点安心踏实地睡觉，到将近七点醒来，窗外早已天黑，泡一杯香浓可口的咖啡，安静读写，在收拾干净的桌前。

好久没有如此惬意安心，在假期无拘无束地做自己，以前总以为一个人独处才能如此，如今与妻相濡以沫，早也可以自在开心地做自己。忘掉曾经看到不知道是谁说过的那句：“一个人永远不可能自由自在地做自己，在另外一个人面前。”

傍晚醒来，躺在床上，忆起学生时代的事，三言两语，与妻说起。本科时，我总会有节律地，在两个星期中找一个下午，在四五点时安静入睡，到七八点醒来，无忧无虑，和室友下楼找点吃的，吃饱回来看看书，或玩玩游戏，晚上几乎在正常作息时间再次入睡。我总觉得这是我最好的休息时光，这样的休息令我身心调节到精力最为舒适、充沛的状态，在一段时间的学习奔忙之后。

一天中不同时间阅读与沉思的体验

发表于 2020-04-27   |   分类于 个人随笔   |  

如今阅读与沉思的时间很分散，不能如学生时代那样整日整日地看书、写作，或是一发不可收拾地思考某个上天入地的问题。学生时代可以睡得很充分，学得很充分，玩得很充分，如今慢慢琐事多了，顾虑多了，休息做事难得尽兴。

今晚写读书笔记，还是用喜欢的每日随身带着的钢笔，一笔一笔写着方块字，心里倍感踏实。不禁想，一天中不同时间段的阅读与沉思所带来的体验是不同的。

夜晚睡前的阅读很放松，一天中再没有任何人、任何事情会来打扰，陷入混沌的梦中世界前，所有的时间都是为我所有的，不再去想任何烦心的事，不再担心突如其来的搅扰，内心平静如水，这是一种一切都将结束前的内心状态，仿佛向死而生，我只把一天当中最后的时间留给自己，做点儿对我来说最心安自在之事，如此方不算虚度了一日。夜晚也是最适合沉思的时间，在经历了一天无意义的时事、新闻、娱乐、公文、八卦、广告这样没有任何营养的、死板的、千篇一律的文字灾难之后，索然无味的语言仿佛才从它本不该待着的使其失去原有魅力的世界中清醒过来，于是我便在夜晚跳脱出来，通过文学作品找到了语言最特别和最有韵味的存在，而此刻头脑对语言和文字又尤其具有灵敏度和辨析能力，能快速理解文学作品所述的含义，在经历一天的俗世沧桑后，特别能理解文字背后表达的含义，找到同感，而这在清晨几乎是很难做到的。夜晚的静和清晨的静不同，夜晚，万物都沉沉沉入了梦乡，这是属于沉思者的静。

雨夜归途

发表于 2020-03-19   |   分类于 个人随笔   |  

其实也说不上雨夜，因为下班那会儿不过傍晚六点半，北辰山小雨生烟，一片朦朦胧胧景象。上班时尽心尽力把事情做好，考虑尽量周全，下班便自在驾驶，回归自我。最近临近单位的山路有一段泥泞，即使小心驾车通过，也还是像在泥里打了个滚，车身从下往上喷出均匀的黄蓝渐变色。虽然周末自己洗车的成果白费了，但静观，仍不失一种另类狂野的美感，黄泥裹着的轮胎给人厚重踏实的感觉，大有“黄沙百战穿金甲”的威严气势！

晚一点返程，错开高峰期，山路便十分安静，没有前后车的追随，可以一路安静地浏览山中风景，音响里伴着轻柔的带点雨声的乐曲，如果天空突然下起雨来，我便会关掉音乐，听雨点打在玻璃窗上滴滴答答的声音，间或传来路上细碎的砂石飞溅起来打到挡泥板上的沙沙声，还有，就是疾驰而过的路面传来的声音。这些声音总给我以漫无目的的思索和平静的浮光掠影的遐想，这时，一天中所有的琐碎与烦恼都消失不见。

寄件

发表于 2020-03-13   |   分类于 诗歌   |  

崎岖山路的转角
偶逢一所孤独的小屋
檐下一盏红灯笼
黎明冷峻的山风从门前扫过

屋外有棵大树
硕大的树冠如一把晴雨伞
伞下立一绿牌，上书
“寄件”二字

空旷的山中人烟稀少
这绿牌惹眼，显得格外孤单
我好奇，究竟多久会有一人为它驻足
寄一件时光的礼物
给心中难以忘却之人
或是年老的自己

相逢在黎明清冷的转角
不经意间
生命所有的历程
我们终将遇到
