回乡前夜

发表于 2022-07-23   |   分类于 诗歌   |  

原以为黑暗是床
漆黑里呼吸
漾出熟悉
闭眼便能入睡

可不恰当时躺下
辗转良久未眠
数心跳——
内在被遗忘的钟

都说孩子学大人
我却要学孩子
倒退，坐下
回到本能
丢开枷锁
回归赤裸、冲动

为好笑的事
笑千百次

相邻几间办公室
年轻战士分散休憩着
廊道有某种沉闷的肃杀
窗外烈日风沙

共同的命运在不远处咆哮

在出征之前——

共赴一场现代战役

躺睡十二个钟头
才重建起身体清晰的水平衡
日子彩色便条般贴满时间之墙
疲惫、困倦、焦躁
子时求索，午时不寐

无知觉的自我使用
消耗的，遗忘的
在一个自然醒来的午后
苏醒

无人区

内裤，短袖短裤
两只袜子，两只鞋
镜片遇阳光渐变墨色的眼镜
手表，帽子
和无名指上烙进落日余晖的戒指
似乎穿上这些重
才够应付工作日一本正经的戏谑
以及夏季冗长白昼的无尽鸣响

傍晚归途转意
跳绿灯时偏航
不顾后视镜里慌乱失控的鸣笛
离开，向卷云与夕阳

未修完的路
成人类文明的遗迹
泊车
在郊野阒静的无人区
摘下帽子，扔掉手表
抖落身上十一片绿叶
沿破败矮墙裸跑
溶进树影黄昏

傍晚爬上天台
信步云端
时有飞机航过
轰隆隆
画一道洁白

卷云天空
如风的冰丝悬缀天幕
开阔清凉
夏日版浓积云像小怪兽
翻滚在远方

阳光穿透冰晶
折射炫彩

我久望太阳的方向
满心期待

期待微笑的环天顶弧
期待日柱
期待天空实验室里
所有魔幻未知的
冰晕传说

观云小感

发表于 2022-06-09   |   分类于 个人随笔   |  

掐指一算，加入总部在英国的赏云协会（Cloud Appreciation Society）已半年有余，平日久坐的脖子和肩颈酸痛得以缓解，因为常常抬头看天，左右前后到处看。记得林夕开玩笑说过，时常看星星可以治好颈椎病！不过，看云一定要注意保护眼睛，白天天空的光线很强，最好戴个防紫外线的墨镜或寻找附近的树木、屋檐、墙等掩体，实在不行就伸手挡挡阳光，避免直视太阳而损伤眼睛。前不久我因每日在山间强光当头的正午看云太久，眼球上出现不少血丝，而且晚上易困，不想再看别的东西，只想好好睡觉休息，应该就是强光对眼睛造成了一些损伤。

这半年的观云实践带给我无限的惊喜和激情。观云实践，我喜欢这么说，因为看云不能光从书本理论上认识云的形成原因和天气条件，还要花大量时间观云，欣赏天空，通过实践来认识天上的各种云彩和光学现象，以及某种云的出现和周围天气的联系。回想起来，我爱上看云，源自小说《岛屿之书》，读到奥克尼群岛仲夏时节持续三周的极昼，午夜的天空偶然会出现“夜光云”。于是搜了下夜光云是什么，接着一系列的延伸网上冲浪：乳状云、贝母云、落幡洞云、飞碟云、山帽云……真是太神奇了，于是一发不可收拾，整日沉迷云中。后来买了赏云协会官方出版物《云彩收集者手册》（The Cloud Collector’s Handbook，由英国的加文·普雷特-平尼著），注册了会员（会员编号：57319），手机下载了App “CloudSpotter”，邮箱每日会收到一张美丽的云朵图片和一段诗意的文字说明。就这样开始了赏云之旅，手册与App相得益彰，中英文介绍了四十多种云彩和大气光学现象，从开始只对几种罕见的云（如荚状云、夜光云、开波、糙面云、马蹄云等）感兴趣，到后来发现每种云奇妙与独特的美。每当看到刚刚习得的新云种出现在天空，我便立马欣喜地打开App拍下提交给协会评审，通过后集云分数和徽章不断增加，全球排行不断靠前，还是挺有乐趣和成就感的。

论勇气与我的生死观

发表于 2022-04-17   |   分类于 个人随笔   |  

今天，第一次带小女儿莎莎逛动物园，在集美的灵玲国际马戏城旁。莎莎再过七天就一周岁两个月啦，最近在学走路，已会咿咿呀呀说话了，小小的脑袋，知道的不少，不仅会认得亲人，家里的电器、日常用品也能记住，还会模仿挺多声音。

以前去马戏团看过表演，欢乐的气氛带给过我快乐。但后来我和梦梦都觉得不忍心再去看此类动物表演，因为训练太残酷，动物很可怜。虽说关在动物园里的动物也失去自由，因人之间的交易而受到伤害，但偶尔与家人、孩子一起看看，亲近动物，认识动物，算是私心下的一种心理妥协吧，毕竟我们大多数人少有野外生存方面的练习，很难真的到纯粹的大自然里去与动物共同生活，近距离观察它们。小朋友与动物是很容易亲近的，他们纯真无邪，自己也是小动物，没有观念，内心纯净，可以原始的方式与动物交流，这点很珍贵，我们应永远如孩子般至简、至真啊。

莎莎刚进动物园那会儿，坐在小推车里睡着了，母亲在前头走，与姐姐的两个小龙凤嘟比、嘟米一起玩，鹦鹉叽叽喳喳，我在后面慢慢推着推车走，陷入一种强烈的情绪当中，这情绪包裹了我一整天。可能是以前看到过这类报道，我在想，如果至为亲近的人不小心落入虎群间，我会怎样？随即便有答案，我将毫不犹豫跟着跳进去，与亲人并肩而战，不论结果。如果犹豫不决或因任何迟疑而胆怯后退，我想我必定会终生抱憾，无法忍受目睹亲人无助、恐惧，在不知所措中面对强敌而死去。落入虎群的含义可以衍生，狮群、豹群、鳄鱼群，恐龙，外星生物，急流旋涡，无边大海，风暴雪崩，人为约定的令人目眩的律法、审判，无法用语言沟通的战争中的异国士兵，或者坠入星空和无尽黑暗的宇宙深渊。

正午，如夜

发表于 2021-10-11   |   分类于 诗歌   |  

天空中
亮白色云朵迅速飘过
风指示方向
在远方山顶聚集
云朵也赶集吗？

正午阳光
投下万物的影子
刨去艳丽芳华的外表
留下本质
留下黑白

一群群骨架在大地绘图
静止的是僵死的建筑原本空洞的结构
动着的是草木的枝干
黑色的叶片、花朵
以及现代人类的骷髅之舞

这大地的黑白
像无声电影
一刻不停在上演
生的世界、死的世界
地平线上共存

表象总是变化，易逝
但本质却延续
大地上的影子不区分时空
事件、样貌、工种、言语、情绪
都荡然无存

唯有静默的黑白永恒
骨架移动，停留
黑色的叶片、花朵
以及黑白如夜的正午
璀璨喧嚣的骷髅之舞

2021.10.11
于北辰山

扯

发表于 2021-10-09   |   分类于 个人随笔   |  

总爱和人开玩笑，又或许认真，说我觉得什么时候应该辞职，辞在灵感枯竭时。判断灵感是否枯竭，很容易，隔一段时间回头看看，成诗几首？是否已到山穷水尽，无词可依、无话可说的境地？

从前我以为，当工作过多地侵蚀生活，过多地占据私人时间，工作得越来越认真投入的时候，便可考虑辞职了，因为工作不过是谋生的手段，是生活可以持续的方式，而非我真正的追求所在，不值得花这许多时间，浪掷青春。但如今我感到，这个因素区居第二。是不会享受闲暇时光了吗？是生活毫无羞耻的重复使我麻木了吗？是荷尔蒙总在深夜勒紧我的脖子让我无心静省吗？是三十岁上有老下有小的日子重压在我肩上吗？是工作定期奖赏的蝇头小利蒙蔽了我的视线吗？是日常人际交往的诸多情绪索萦心间吗？这些也许值得反思，但只要我的灵感还未枯竭，仅有的不多的高光时刻偶尔降临，我便满心欢喜。

#张弛 #诗意

止境，适境

发表于 2021-10-08   |   分类于 诗歌   |  

我在此山中打井
你在别处打
虽周遭风景迥异
然同样用工
地底水源终将我们连通
正如夜里仰望天空
明月的美映在不同地域人们的眼中

我没有语言天赋
英语学习多年
仍止于纸面的读懂
若赏英文诗
很难由一行行字母联想万物
而简体中文
时常品着便神游物外

也如自学音乐
记得五线谱上音符对应的琴键
却难望谱而生乐音
耳畔浮不起旋律
唯看得音符纷纷跃然纸上
肆意飞舞
离我而去

时光在风起的书页间流走
大师大作浩如烟海
年逾三十
更钟一隅之静
一瓢之饮
一言之美
享心中所乐

于北辰山
